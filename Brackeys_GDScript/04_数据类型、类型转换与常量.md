# 04 数据类型、类型转换与常量

## 作用域

声明变量时要考虑**在哪里声明**：

- 在 `if` 语句或函数**内部**声明的变量，只能在该 `if` 或该函数内部使用。这称为**作用域（scope）**，是初学者常犯错的地方。
- 如果希望在脚本的不同位置使用某个变量，就把它放在**顶部、任何函数之外**，成为脚本范围的变量，可在脚本任何地方访问。

```gdscript
var script_var = 1     # 脚本范围，处处可用

func _ready():
    var local_var = 2  # 只能在 _ready() 内使用
```

## 动态类型与数据类型

GDScript 默认是**动态类型**语言：声明变量时不必指定它能持有哪种类型，甚至可以重新赋值改变其数据类型。

```gdscript
var coolness = 100
coolness = true        # 合法：动态类型允许改变所持有的数据类型
```

这让创建变量快速灵活，但也更容易出错、性能不如静态类型。

四种经典数据类型：

| 类型 | 说明 |
|------|------|
| `bool`（Boolean） | `true` / `false` |
| `int`（Integer） | 整数 |
| `float` | 带小数的数字 |
| `String` | 文本 |

## 类型转换（casting）

把一种类型转换为另一种类型称为**类型转换**。

```gdscript
var number = 42
var text = "答案是 "
print(text + str(number))   # 用 str() 把 int 转成 String

var pi = 3.14
print(int(pi))              # 3
```

> 注意：`int()` 会**直接截断**小数部分，而不是四舍五入，因此 `3.9` 也会得到 `3`。

## Vector2 与 Vector3

`Vector2` 存两个 float（`x`、`y`），`Vector3` 存三个 float（`x`、`y`、`z`），最常用于表示位置。可以单独修改其中一个分量：

```gdscript
var position = Vector3(3, -10, 5)
position.x += 2
print(position)             # (5, -10, 5)
```

## 静态类型

GDScript 允许对任意变量做**静态类型**指定，按需选择：

```gdscript
var damage: int = 15        # 显式指定类型
var damage := 15            # 推断类型（结果相同，仍是静态 int）
```

Godot 识别出 `15` 是整数，从而推断变量为 `int`。此后该变量不能再改为其他类型，若尝试赋字符串会报错。

## @export

在变量前加 `@export` 注解，就能在 **Inspector（检查器）** 中设置它；脚本中写的值只作为默认值。

```gdscript
@export var damage: int = 15
```

保存脚本后选中节点，即可在检查器里设值；点击圆形箭头可恢复默认值。

## 常量 const

需要一个**不可更改**的值时，用 `const` 定义常量。惯例上以全大写命名：

```gdscript
const GRAVITY = -9.81
```

常量可以像其他变量一样使用，但**不能被修改**，否则会抛出错误。
