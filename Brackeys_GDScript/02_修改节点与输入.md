# 02 修改节点与输入

## 修改节点 1.0

要通过脚本修改一个节点，需要两样东西：**对节点的引用** + **要修改的属性名**。

### 准备一个节点

1. 添加一个 **Label** 节点，居中放置，填入默认文本，设置为水平/垂直居中，并覆盖字体大小。
2. 把鼠标悬停在 Inspector 的 **Text** 属性上，可以看到脚本中访问它的名称是小写的 `text`。

### 获取引用并修改属性

在脚本里获取引用最简单的方式：**把节点从场景树拖进脚本**，会生成 `$节点名`。再用点号访问属性：

```gdscript
func _ready():
    $Label.text = "hello world"
```

运行后，标签文本变为 `hello world`。

### 用 modulate 着色

`modulate` 属性可以给精灵和 UI 元素着色（在 Inspector 的 **Visibility** 分组下）。

```gdscript
func _ready():
    $Label.text = "hello world"
    $Label.modulate = Color.GREEN
```

`Color` 提供了一系列内置颜色常量（如 `Color.GREEN`、`Color.RED`）。

## 输入

目标：**按下空格键时把标签变红，松开时变回绿色**。

### 设置输入动作

1. 打开 **项目 → 项目设置 → 输入映射（Input Map）**。
2. 添加一个动作。动作用于把按键绑定到我们希望发生的事件上（常见如射击、跳跃）。这里命名为 `my_action`。
3. 点击**添加**，再点加号，按下**空格键**并确定，即可把空格键绑定到 `my_action`。

### 用 `_input()` 响应输入

`_input()` 是 Godot 的内置函数：它不是在游戏开始时调用，而是**每当游戏接收到任何输入时**运行。参数 `event` 是触发该函数的输入信息（鼠标移动、按键等）。

```gdscript
func _input(event):
    if event.is_action_pressed("my_action"):
        $Label.modulate = Color.RED
    if event.is_action_released("my_action"):
        $Label.modulate = Color.GREEN
```

- `event.is_action_pressed("my_action")`：动作被**按下**时为真。
- `event.is_action_released("my_action")`：动作被**释放**时为真。

运行游戏，按下空格键标签变红，松开变回绿色。

> 这只是 Godot 中处理输入的多种方式之一。本教程聚焦 GDScript 语言，输入的更多内容可另行查阅官方文档。
