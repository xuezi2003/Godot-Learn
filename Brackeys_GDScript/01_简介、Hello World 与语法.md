# 01 简介、Hello World 与语法

## GDScript 简介

**GDScript** 是一种高级、面向对象、命令式、渐进类型（gradually typed）的编程语言，其语法与 **Python** 非常相似，专为 **Godot 游戏引擎**设计。

- 本教程更像是一份**概览**，而不是逐步跟做的项目教程，适合当作日后的参考手册。
- 初学者不必一次性理解所有内容，随时可以暂停、动手尝试，之后再回来继续。
- 本视频讲的是 **GDScript 语言**，而不是 Godot 入门。如果你从未用过 Godot，建议先看《在 Godot 中制作你的第一个游戏》。

## Hello World

### 创建测试场景与脚本

1. 在一个空项目中点击加号，**添加一个节点**，选择基础的 **Node**，重命名为 `Main`。
2. 保存场景（`Ctrl + S`）。
3. 点击节点旁的**卷轴图标**为选中的节点创建脚本，使用**默认模板**，脚本名默认为 `main.gd`（首字母小写）。

脚本默认包含两个函数，我们先关注顶部的 `_ready()`。

### `_ready()` 函数

- 当节点**首次进入场景树**时被调用，可以理解为「游戏开始的时刻」。
- 因此适合放置希望立即执行的初始化代码。

### `pass` 关键字

- 新建的函数里只有一个 `pass`。
- `pass` 表示**不执行任何操作**，用于占位，避免函数体为空而报错。

### 打印消息

用打印代码替换 `pass`：

```gdscript
func _ready():
    print("hello world")
```

### 运行与停止

- 按下右上角的运行按钮或 `F5` 运行游戏；若尚未设置主场景，会询问运行哪个场景，选择**当前场景**即可。
- **输出窗口（控制台）** 会显示打印信息，也是查看错误的地方。
- 按下停止按钮或 `F8` 停止运行。

## 语法

编写 GDScript 时有几点需要特别注意：

### 以换行结束，无需分号

- 语句以**换行符**结束，末尾**不需要**添加分号（`;`）。

### 用缩进确定代码结构

- 与 Python 类似，GDScript 使用**缩进（Tab）** 来确定代码结构，而不是像基于 C 的语言那样使用花括号。
- 缩进告诉 Godot 代码「属于哪里」。下例中，`print` 前的一个 Tab 表示它属于 `_ready()` 函数：

```gdscript
func _ready():
    print("hello world")  # 缩进表示这行属于 _ready()
```

- 如果删掉这个 Tab，就会报错。

### 区分大小写

- GDScript **区分大小写**。例如把 `print` 写成大写的 `Print` 会报错。
