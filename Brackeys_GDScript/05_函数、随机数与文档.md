# 05 函数、随机数与文档

## 函数

**函数**是编程的基础，它让你把代码组织成小而可复用的单元。

我们一直在用的 `_ready()`、`_input()` 都是 Godot 的内置函数，它们前面带**下划线**，表示由引擎自身调用，而不是由我们主动调用。我们也可以创建自己的函数（游戏中常见的如「跳跃」「死亡」「射击」「重生」）。

### 创建与调用

用 `func` 定义函数，写上名称、圆括号和冒号：

```gdscript
func jump():
    print("jump")

func _input(event):
    if event.is_action_pressed("my_action"):
        jump()          # 用圆括号调用函数
```

### 参数与返回值

函数可以有输入和输出：给函数的输入称为**参数（parameter）**，输出称为**返回值（return value）**。

```gdscript
func add(num1, num2):
    var result = num1 + num2
    return result       # 用 return 把结果返回给调用处

func _ready():
    var result = add(3, 5)
    print(result)       # 8
    result = add(result, 10)
    print(result)       # 18
```

只有用 `return` 返回结果，才能在调用处（如 `_ready()`）拿到并继续使用它。

### 静态类型化参数与返回值

和变量一样，可以给参数和返回值指定类型（用箭头 `->` 指定返回类型）：

```gdscript
func add(num1: int, num2: int) -> int:
    return num1 + num2
```

这样该函数就明确只用于处理整数。

## 随机数

- `randf()`：返回 `0.0` 到 `1.0` 之间的随机浮点数，适合分配概率。

```gdscript
var roll = randf()
if roll <= 0.8:
    print("你找到了一件普通物品")   # 约 80% 概率
else:
    print("你找到了一件稀有物品")   # 约 20% 概率
```

- `randf_range(min, max)` / `randi_range(min, max)`：返回最小值与最大值之间的随机浮点数 / 整数。

```gdscript
var character_height = randi_range(140, 210)
print("你的角色是 " + str(character_height) + " 厘米高")
```
