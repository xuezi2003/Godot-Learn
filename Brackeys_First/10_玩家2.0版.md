# 10 玩家 2.0 版

## 本节目标

- 理解 `physics_process` 与 `process` 的区别
- 重新绑定输入按键为自定义动作
- 根据移动方向翻转玩家精灵
- 为玩家添加奔跑和跳跃动画

## 玩家移动模板解析

- 脚本顶部定义了速度、跳跃速度等常量。
- `gravity` 变量根据项目设置默认为 980。
- 核心逻辑在 `_physics_process(delta)` 函数中。

### `process` 与 `physics_process`

| 函数 | 调用频率 | 用途 |
|------|---------|------|
| `_process(delta)` | 每帧调用，跟随实际帧率 | 视觉表现、非物理逻辑 |
| `_physics_process(delta)` | 固定频率调用，默认每秒 60 次 | 移动、跳跃、碰撞等物理相关逻辑 |

- 物理运算需要固定更新间隔，避免卡顿和不稳定。
- 玩家角色涉及碰撞和重力，因此使用 `_physics_process`。

### 模板核心逻辑

1. 如果玩家不在地面上，则施加重力。
2. 如果按下跳跃键且玩家在地面上，则跳跃。
3. 根据左右输入计算方向并移动。

## 重新绑定输入

### 输入映射

- Godot 使用**动作（Action）**系统管理输入。
- 可以为跳跃、左右移动等创建动作，再将按键绑定到动作上。
- 一个动作可以绑定多个按键。

### 设置自定义动作

1. 打开 **Project > Project Settings > Input Map**。
2. 添加以下动作：
   - `jump`
   - `move_left`
   - `move_right`
3. 为每个动作绑定按键：
   - `jump`：空格键
   - `move_left`：左方向键、A 键
   - `move_right`：右方向键、D 键
4. 在脚本中将默认的 `ui_*` 动作替换为自定义动作：
   - 跳跃：`ui_accept` → `jump`
   - 左移：`ui_left` → `move_left`
   - 右移：`ui_right` → `move_right`

## 根据方向翻转精灵

1. 获取 `AnimatedSprite2D` 引用：

```gdscript
@onready var animated_sprite = $AnimatedSprite2D
```

2. 在 `_physics_process` 中获取输入方向：

```gdscript
var direction = Input.get_axis("move_left", "move_right")
```

- 不按键时 `direction` 为 0。
- 按右键时为 1，按左键时为 -1。

3. 根据方向设置水平翻转：

```gdscript
if direction > 0:
    animated_sprite.flip_h = false
elif direction < 0:
    animated_sprite.flip_h = true
```

## 添加奔跑与跳跃动画

### 添加动画

1. 选中 `AnimatedSprite2D`。
2. 在 SpriteFrames 面板中添加两个新动画：
   - **Run**（奔跑）
   - **Jump**（跳跃）
3. 为 **Run** 添加骑士的奔跑帧（跨越两行），FPS 设为 10。
4. 为 **Jump** 添加一个合适的帧（例如翻滚动画中的一帧）。

### 播放动画

```gdscript
if is_on_floor():
    if direction == 0:
        animated_sprite.play("Idle")
    else:
        animated_sprite.play("Run")
else:
    animated_sprite.play("Jump")
```

- `is_on_floor()` 是 CharacterBody2D 内置函数，检测是否站在地面上。
- 在地面上时，根据是否移动播放 Idle 或 Run。
- 在空中时播放 Jump。

## 完整玩家脚本示例

```gdscript
extends CharacterBody2D

const SPEED = 130.0
const JUMP_VELOCITY = -300.0

@onready var animated_sprite = $AnimatedSprite2D

func _physics_process(delta):
    if not is_on_floor():
        velocity += get_gravity() * delta

    if Input.is_action_just_pressed("jump") and is_on_floor():
        velocity.y = JUMP_VELOCITY

    var direction = Input.get_axis("move_left", "move_right")

    if direction > 0:
        animated_sprite.flip_h = false
    elif direction < 0:
        animated_sprite.flip_h = true

    if is_on_floor():
        if direction == 0:
            animated_sprite.play("Idle")
        else:
            animated_sprite.play("Run")
    else:
        animated_sprite.play("Jump")

    if direction:
        velocity.x = direction * SPEED
    else:
        velocity.x = move_toward(velocity.x, 0, SPEED)

    move_and_slide()
```
