# 08 敌人

## 本节目标

- 创建史莱姆敌人场景
- 复用死亡区域场景
- 使用脚本控制敌人在两堵墙之间来回移动
- 使用 RayCast2D 检测墙壁
- 根据移动方向翻转敌人图像

## 创建敌人场景

1. 新建场景，根节点选择 **Node2D**。
2. 将根节点重命名为 **Slime**。
3. 保存到 `scenes` 文件夹，命名为 `slime.tscn`。

### 添加敌人动画

1. 添加子节点 **AnimatedSprite2D**。
2. 为 Sprite Frames 创建 **New SpriteFrames**。
3. 从精灵表加载 `assets/sprites/slime_green.png`（绿色史莱姆）。
4. 精灵表尺寸为 **4×3**。
5. 该资源包含三个动画：苏醒、待机、受伤。
6. 选择待机动画的 4 帧，添加到 SpriteFrames 中。
7. 将 FPS 设置为 **10**，启用自动播放。
8. 将精灵向上移动一点，使其站在地面上。

### 复用死亡区域

1. 在 `slime` 场景中实例化 `killzone.tscn`。
2. 选中 `Killzone` 实例，添加子节点 **CollisionShape2D**。
3. Shape 选择 **RectangleShape2D**，调整大小覆盖史莱姆身体。
4. 按住 `Alt` 键可等比例缩放。

### 完成场景

1. 将根节点重命名为 **Slime**。
2. 保存场景。
3. 将 `slime.tscn` 拖入 `game` 场景并运行。
4. 玩家碰到史莱姆会触发死亡区域，游戏重启。

## 敌人移动脚本

1. 在 `slime` 场景中选中 `Slime` 节点。
2. 添加脚本，选择默认模板，保存到 `scripts/slime.gd`。
3. 删除 `_ready()` 函数，保留 `_process(delta)`。

### 游戏循环与 `process`

- 游戏通过不断循环更新状态并绘制帧来运行。
- `_process(delta)` 每帧调用一次，适合处理随时间变化的逻辑。
- `delta` 是自上一帧以来经过的时间（秒）。
- 将速度乘以 `delta`，可以使移动速度不受帧率波动影响。

### 基础移动

```gdscript
const SPEED = 60

func _process(delta):
    position.x += SPEED * delta
```

- 这样敌人会以每秒 60 像素的速度向右移动。

### 添加方向控制

```gdscript
const SPEED = 60
var direction = 1

func _process(delta):
    position.x += direction * SPEED * delta
```

- `direction` 为 1 时向右，为 -1 时向左。

## 检测墙壁

### RayCast2D

- **RayCast2D** 是用于检测碰撞的不可见射线。
- 可以检测射线方向上是否有物体。

1. 在 `slime` 场景中添加子节点 **RayCast2D**。
2. 将射线起点移到史莱姆中心，朝右发射。
3. 重命名为 **RayCastRight**。
4. 按 `Ctrl + D` 复制该节点。
5. 将复制的射线朝左发射，重命名为 **RayCastLeft**。

### 获取射线引用

1. 在脚本顶部，同时选中两个 RayCast2D 节点。
2. 拖入代码编辑器，释放时按住 `Ctrl` 键。
3. 自动生成引用：

```gdscript
@onready var ray_cast_right = $RayCastRight
@onready var ray_cast_left = $RayCastLeft
```

### 根据碰撞改变方向

```gdscript
func _process(delta):
    if ray_cast_right.is_colliding():
        direction = -1
    if ray_cast_left.is_colliding():
        direction = 1

    position.x += direction * SPEED * delta
```

- 右侧检测到墙壁时向左走。
- 左侧检测到墙壁时向右走。
- 敌人会在两堵墙之间自动往返移动。

## 翻转精灵

1. 选中 `AnimatedSprite2D`，在 Inspector 的 **Offset** 中找到 **Flip H**（水平翻转）。
2. 获取动画精灵引用：

```gdscript
@onready var animated_sprite = $AnimatedSprite2D
```

3. 在改变方向时同步设置翻转：

```gdscript
func _process(delta):
    if ray_cast_right.is_colliding():
        direction = -1
        animated_sprite.flip_h = true
    if ray_cast_left.is_colliding():
        direction = 1
        animated_sprite.flip_h = false

    position.x += direction * SPEED * delta
```

## 完整脚本示例

```gdscript
extends Node2D

const SPEED = 60
var direction = 1

@onready var animated_sprite = $AnimatedSprite2D
@onready var ray_cast_right = $RayCastRight
@onready var ray_cast_left = $RayCastLeft

func _process(delta):
    if ray_cast_right.is_colliding():
        direction = -1
        animated_sprite.flip_h = true
    if ray_cast_left.is_colliding():
        direction = 1
        animated_sprite.flip_h = false

    position.x += direction * SPEED * delta
```
