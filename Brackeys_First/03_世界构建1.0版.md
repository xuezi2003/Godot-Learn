# 03 世界构建 1.0 版

## 本节目标

- 使用 TileMap 创建 2D 关卡
- 配置瓦片集（TileSet）并绘制地图
- 为瓦片添加物理碰撞
- 让摄像机跟随玩家移动

## 移除临时地面

- 删除之前用于测试的 StaticBody2D 地面。
- 将使用 TileMap 替代，构建真正的可站立地面。

## 瓦片地图基础

- **TileMap**：用于在网格上绘制瓦片的节点，是 2D 游戏构建关卡最常用的方式。
- **TileSet**：瓦片资源库，包含所有可用瓦片及其属性。
- 通过少量瓦片组合，可以创造出丰富多变的关卡。

## 创建 TileMap

1. 在 `game` 场景中添加 **TileMap** 节点。
2. 在 Inspector 中为 **Tile Set** 创建 **New TileSet**。
3. 确保 **Tile Size** 设置为 **16×16**（与资源匹配）。

## 配置瓦片集

1. 切换到编辑器底部的 **TileSet** 标签页。
2. 将 `assets/sprites/world_tileset.png` 拖入 TileSet 面板。
3. 弹出提示时选择**自动创建瓦片**。
4. Godot 会自动检测网格并切分瓦片。
5. 使用橡皮擦删除自动识别错误的多余瓦片（例如棕榈树顶部被拆分成多个小瓦片）。
6. 按住 `Shift` 点击并拖动，可将分散的瓦片合并为一个完整大瓦片。
7. 树木可被保留为多个瓦片，以便创建不同高度的树。

## 绘制关卡

1. 切换到 **TileMap** 标签页。
2. 选择绘制工具，从瓦片集中选择瓦片开始绘制。
3. 使用矩形工具可快速填充大片区域。
4. 右键可擦除瓦片。
5. 先勾勒关卡的基本结构，留出放置平台、敌人等对象的空间，最后再进行装饰。
6. 可以一次选择多个瓦片同时绘制。
7. 使用选择模式可以选中并移动大块瓦片。

### 绘制树木技巧

- 先绘制树干底部。
- 继续向上绘制树干中间部分到想要的高度。
- 最后绘制树冠顶部。

## 添加物理碰撞

1. 运行游戏，玩家仍会穿过瓦片，因为瓦片还没有碰撞。
2. 在 TileSet 面板中找到 **Physics Layers**，添加一个新的物理层级。
3. 选择需要产生碰撞的实心瓦片（地面、平台等）。
4. 在瓦片上绘制物理碰撞形状。
5. 树木、灌木等不需要碰撞的瓦片保持空白。

### 擦除与恢复

- 点击清除按钮，可以绘制无物理效果的瓦片。
- 按 `F` 键恢复默认的完整瓦片碰撞形状。
- 快捷键：`C` 清除，`F` 恢复默认。

### 桥等特殊瓦片

1. 桥梁只需要一部分有碰撞。
2. 选中桥梁瓦片，在左侧调整碰撞形状的点。
3. 使用鼠标左键添加点，右键删除点。
4. 为三段桥梁分别配置合适的碰撞多边形，确保接合处精确，避免角色卡住。

## 摄像机跟随

1. 将 `TileMap` 节点拖到场景树最上方，便于查找和管理。
2. 在 `Player` 节点下添加 **Camera2D**（如果尚未添加）。
3. 将 Camera2D 放在玩家正上方。
4. 启用 **Position Smoothing**，默认速度即可。
5. 运行游戏，摄像机平滑跟随玩家移动。
