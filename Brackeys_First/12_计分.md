# 12 计分

## 本节目标

- 创建 GameManager 管理全局游戏数据
- 实现分数增加函数
- 使用唯一名称访问跨节点引用
- 在 Label 中显示当前分数

## 创建 GameManager

1. 在 `game` 场景中添加一个普通 **Node** 节点。
2. 重命名为 **GameManager**。
3. 放在场景树最上方，便于管理。
4. 使用普通 Node 而非 Node2D，因为 GameManager 不需要位置、旋转、缩放等变换信息。

### 添加脚本

1. 选中 `GameManager`，添加脚本，选择空模板。
2. 保存到 `scripts/game_manager.gd`。

### 分数变量

```gdscript
var score = 0
```

### 加分函数

```gdscript
func add_point():
    score += 1
    print(score)
```

- 使用 `func` 关键字定义函数。
- `score += 1` 将分数加 1。
- 当前只是打印分数，还没有显示在屏幕上。

## 从金币调用加分函数

### 问题

- `GameManager` 在场景树中比金币层级更高。
- 直接用路径引用会导致不稳定的路径字符串。

### 唯一名称访问

1. 右键 `GameManager` 节点，选择 **作为唯一名称访问（Access as Unique Name）**。
2. 节点旁会出现百分比图标 `%`，表示这是唯一节点。
3. 将 `GameManager` 拖入 `coin.gd` 时，会自动生成 `%GameManager` 引用。

```gdscript
@onready var game_manager = %GameManager
```

### 限制

- 唯一名称只在同一场景内有效。
- 如果节点位于不同场景，无法通过唯一名称访问。

### 调用加分

```gdscript
func _on_body_entered(body):
    game_manager.add_point()
    queue_free()
```

- 拾取金币时调用 `GameManager.add_point()`，分数增加。

## 显示分数

1. 在 `game` 场景中添加一个 **Label**，命名为 **ScoreLabel**。
2. 调整位置、字体、颜色和对齐方式。
3. 将 **Autowrap Mode** 设为 **Word**，启用自动换行。
4. 将水平对齐设为居中。
5. 将 `ScoreLabel` 拖到 `GameManager` 节点下，便于从脚本引用。

### 获取 Label 引用

```gdscript
@onready var score_label = $ScoreLabel
```

### 更新 Label 文本

```gdscript
func add_point():
    score += 1
    score_label.text = "you collected " + str(score) + " coins"
```

- `str(score)` 将整数转换为字符串。
- 使用 `+` 连接字符串。

## 完整 GameManager 脚本

```gdscript
extends Node

var score = 0

@onready var score_label = $ScoreLabel

func add_point():
    score += 1
    score_label.text = "you collected " + str(score) + " coins"
```
