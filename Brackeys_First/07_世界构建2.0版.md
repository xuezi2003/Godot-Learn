# 07 世界构建 2.0 版

## 本节目标

- 整理场景树，使用空节点对对象进行分类
- 为 TileMap 添加背景图层
- 继续扩展和美化关卡

## 整理场景节点

1. 打开 `game` 场景。
2. 添加一个普通 **Node** 节点作为分类容器。
3. 重命名为 **Coins**，将所有金币实例拖到该节点下。
4. 同样创建 **Platforms** 节点，将所有平台实例拖入其中。
5. 对场景中其它杂乱的元素也进行类似归类。
6. 这样可以使场景树更清晰，便于管理。

## 添加 TileMap 背景图层

1. 选中 `TileMap` 节点。
2. 在 Inspector 的 **Layers** 中添加一个新的图层。
3. 将新图层重命名为 **Background**（背景）。
4. 将原有的图层重命名为 **Midground**（中景）。
5. 将 **Background** 图层拖到最上方，确保它最先被绘制。

### 绘制背景

1. 在 TileMap 编辑器中选择 **Background** 图层。
2. 从 TileSet 中选择背景瓦片开始绘制。
3. 使用矩形工具可以一次性填充大片背景区域。
4. 背景图层不会被玩家碰撞，仅用于视觉装饰。

## 扩展关卡

1. 根据已有元素继续扩展关卡结构。
2. 添加更多游戏性元素和装饰。
3. 不断尝试和修改，直到对关卡布局满意。

## 注意事项

- 图层顺序决定绘制顺序，先绘制的会被后绘制的覆盖。
- 中景（地面、平台等）应在背景之上绘制。
- 使用分类节点可以保持场景树整洁，尤其在对象数量增多时非常重要。
