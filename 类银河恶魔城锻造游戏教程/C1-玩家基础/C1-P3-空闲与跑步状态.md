# C1-P3：空闲与跑步状态

> 本集完善 `Idle` 与 `Run` 状态，修复输入漂移问题，并通过导出变量在编辑器中调整移动速度。

## 1. 课后作业回顾

### 1.1 Mono 的思路

- 在 `Idle.handle_input` 中检测左右按键按下，切换到 `Run`。
- 在 `Run.handle_input` 中检测按键释放，切回 `Idle`。

### 1.2 Aya 的思路

- 在 `Idle.physics_process` 中检测 `direction.x != 0`，返回 `Run`。
- 在 `Run.physics_process` 中检测 `direction.x == 0`，返回 `Idle`。
- 还实现了旋转速度，让角色平滑贴合斜坡角度。

> 本集采用 `physics_process` 检测方向的方式实现切换。

## 2. 修复输入漂移

### 2.1 问题

`Input.get_vector("left", "right", "up", "down")` 的死区处理有个缺陷：只要某个轴向超过死区，所有轴向的微小值都会被保留，导致摇杆漂移。

### 2.2 解决：按轴向单独计算

```gdscript
func update_direction() -> void:
    var previous_direction := direction
    direction = Vector2(
        Input.get_axis("left", "right"),
        Input.get_axis("up", "down")
    )
```

- `Input.get_axis("left", "right")` 会自动应用死区过滤。
- 分别计算 X、Y 轴，避免漂移污染另一轴。
- > 项目设置中输入轴的默认死区值为 **0.2**。这意味着摇杆输入绝对值小于 0.2 时会被忽略，只有超过该阈值才会产生有效输入。

## 3. 状态引用

在 `PlayerState` 基类中声明：

```gdscript
@onready var idle: PlayerState = %Idle
@onready var run: PlayerState = %Run
```

- 右键状态节点 → 勾选「以唯一名称访问」。
- 拖拽到脚本中即可生成 `@onready` 引用。

## 4. 状态标签调试

- 在玩家场景中添加 `Label` 节点，显示当前状态名。
- 在 `change_state` 中更新标签文本：

```gdscript
state_label.text = current_state.name
```

- 这样运行时头顶会显示 `Idle` / `Run` 等状态。

## 5. Idle 状态

```gdscript
func process(_delta: float) -> PlayerState:
    if player.direction.x != 0:
        return run
    return null

func physics_process(_delta: float) -> PlayerState:
    player.velocity.x = 0
    if not player.is_on_floor():
        return fall
    return null
```

- 待机时水平速度归零。
- 不在地面时进入 `Fall`。

## 6. Run 状态

```gdscript
func process(_delta: float) -> PlayerState:
    if player.direction.x == 0:
        return idle
    if player.direction.y >= 0.5:
        return crouch
    return null

func physics_process(delta: float) -> PlayerState:
    player.velocity.x = player.direction.x * player.move_speed
    if not player.is_on_floor():
        return fall
    return null
```

- 奔跑时水平速度 = 方向 × 移动速度。
- 松开水平输入时回到 `Idle`。
- 按下下方向时进入 `Crouch`。

## 7. 导出变量 `move_speed`

```gdscript
# player.gd
@export var move_speed: float = 150.0
```

- 在编辑器检查器中可直接调整。
- 默认值 `150`。
- 点击属性旁的重置按钮可恢复默认值。
- > `@export` 变量的默认值由脚本中的赋值决定；在检查器中修改后，点击重置按钮就会回到脚本里写的默认值。
