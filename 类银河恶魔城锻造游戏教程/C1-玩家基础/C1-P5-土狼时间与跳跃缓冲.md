# C1-P5：土狼时间与跳跃缓冲

> 为跳跃系统加入土狼时间（Coyote Time）和跳跃缓冲（Jump Buffer），让平台操作更宽容、更有响应感。

## 1. 土狼时间（Coyote Time）

### 1.1 概念

- 玩家离开平台边缘后，短时间内仍可起跳。
- 名称源自卡通中 Wiley Coyote 冲出悬崖后不会立刻下坠的效果。

### 1.2 实现

在 `fall_state.gd` 中：

```gdscript
@export var coyote_time: float = 0.125
var coyote_timer: float = 0.0

func enter() -> void:
    if player.previous_state != jump:
        coyote_timer = coyote_time
    else:
        coyote_timer = 0.0

func process(delta: float) -> PlayerState:
    coyote_timer -= delta
    return null

func handle_input(event: InputEvent) -> PlayerState:
    if event.is_action_pressed("jump") and coyote_timer > 0:
        return jump
    return null
```

### 1.3 关键点

- 只有从非 `Jump` 状态进入 `Fall` 时才重置土狼计时器（即自然掉落）。
- 从 `Jump` 进入 `Fall` 时不重置，避免无限跳跃。
- 推荐值：`0.1 ~ 0.2` 秒，作者常用 `0.125`。

## 2. 下落重力增强

### 2.1 目的

- 让下落速度略快于上升速度，增加重量感与利落感。

### 2.2 实现

在 `player.gd` 中：

```gdscript
var gravity_multiplier: float = 1.0
```

重力计算改为：

```gdscript
velocity.y += gravity * gravity_multiplier * delta
```

在 `fall_state.gd` 中：

```gdscript
@export var fall_gravity_multiplier: float = 1.165

func enter() -> void:
    player.gravity_multiplier = fall_gravity_multiplier

func exit() -> void:
    player.gravity_multiplier = 1.0
```

- 进入下落时增加重力倍数。
- 退出下落时重置。
- > `1.165` 这个值来自对《超级马力欧兄弟》跳跃手法的分析：下落重力略大于上升重力，能让跳跃更有重量感和利落感。

## 3. 跳跃缓冲（Jump Buffer）

### 3.1 概念

- 玩家在落地前瞬间按下跳跃键，落地后会立即起跳。
- 解决「按键稍早导致没跳起来」的挫败感。

### 3.2 实现

在 `fall_state.gd` 中：

```gdscript
@export var jump_buffer_time: float = 0.2
var buffer_timer: float = 0.0

func handle_input(event: InputEvent) -> PlayerState:
    if event.is_action_pressed("jump"):
        if coyote_timer > 0:
            return jump
        buffer_timer = jump_buffer_time
    return null

func process(delta: float) -> PlayerState:
    coyote_timer -= delta
    buffer_timer -= delta
    return null

func physics_process(_delta: float) -> PlayerState:
    if player.is_on_floor():
        if buffer_timer > 0:
            return jump
        return idle
    return null
```

### 3.3 流程

1. 玩家在空中按下跳跃 → `buffer_timer` 重置为 `0.2`。
2. `buffer_timer` 每帧递减。
3. 落地瞬间若 `buffer_timer > 0`，立即进入 `Jump`。

## 4. 本集完成效果

- 离开平台边缘后仍可短暂起跳。
- 落地前提前按跳会自动衔接下一次跳跃。
- 下落速度略快于上升，操作更有重量感。
