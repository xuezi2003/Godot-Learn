# C1-P4：跳跃与坠落状态

> 本集实现 `Jump` 与 `Fall` 两个状态，加入基础可变高度跳跃，并添加可视化调试指示器。

## 1. 课后作业回顾

### 1.1 Pigeon 的方案

- 使用 `Timer` 控制跳跃持续时间。
- 进入 `Jump` 时创建计时器，超时后进入 `Fall`。

### 1.2 James 的方案

- 实现了可变跳跃高度，效果流畅。

### 1.3 作者的思路

- 不基于计时器，而是基于速度实现跳跃。
- 将跳跃拆分为 `Jump`（上升）和 `Fall`（下降）两个状态。

## 2. 为什么拆分 Jump / Fall

- 从平台边缘走出时可直接进入 `Fall`，无需先 `Jump`。
- 每个状态的职责更清晰。
- 便于后续分别处理土狼时间、跳跃缓冲、下落动画等。

## 3. 添加 Jump / Fall 状态节点

```
States/
├── Idle
├── Run
├── Jump
└── Fall
```

- 每个节点挂载独立的 `jump_state.gd` / `fall_state.gd`，继承 `PlayerState`。
- 在 `PlayerState` 中添加引用：

```gdscript
@onready var jump: PlayerState = %Jump
@onready var fall: PlayerState = %Fall
```

## 4. 进入跳跃状态

### 4.1 在 Idle / Run 中监听跳跃输入

```gdscript
func handle_input(event: InputEvent) -> PlayerState:
    if event.is_action_pressed("jump"):
        return jump
    return null
```

### 4.2 Jump 状态进入逻辑

```gdscript
@export var jump_speed: float = 450.0

func enter() -> void:
    player.velocity.y = -jump_speed
```

> 注意：这里用 `=` 直接赋值，而不是 `-=`。因为后续二段跳需要完整初速度。

## 5. Jump 状态退出条件

```gdscript
func physics_process(_delta: float) -> PlayerState:
    player.velocity.x = player.direction.x * player.move_speed
    if player.is_on_floor():
        return idle
    if player.velocity.y >= 0:
        return fall
    return null
```

- 持续允许空中水平移动。
- 若落地则回到 `Idle`。
- 若速度方向转为向下，则进入 `Fall`。

## 6. Fall 状态

```gdscript
func physics_process(_delta: float) -> PlayerState:
    player.velocity.x = player.direction.x * player.move_speed
    if player.is_on_floor():
        return idle
    return null
```

- 下落时同样允许空中水平移动。
- 落地后回到 `Idle`。

## 7. 从边缘掉落

在 `Idle` 和 `Run` 的 `physics_process` 中添加：

```gdscript
if not player.is_on_floor():
    return fall
```

- 这样走出平台边缘时会自动进入 `Fall`。

## 8. 可变高度跳跃

### 8.1 实现思路

- 当玩家松开跳跃键时，削减向上的速度，使跳跃变短。

```gdscript
func handle_input(event: InputEvent) -> PlayerState:
    if event.is_action_released("jump"):
        player.velocity.y *= 0.5
        return fall
    return null
```

### 8.2 效果

- 长按：完整高度跳跃。
- 轻点：速度减半，角色提前下落，形成短跳。

## 9. 调试指示器

### 9.1 创建场景

- 新建 `debug_jump_indicator.tscn`，根节点为 `Node2D`。
- 添加小圆点和短线作为视觉标记。

### 9.2 实例化函数

```gdscript
const DEBUG_JUMP_INDICATOR = preload("res://Player/debug_jump_indicator.tscn")

func add_debug_indicator(color: Color) -> void:
    var d := DEBUG_JUMP_INDICATOR.instantiate()
    get_tree().root.add_child(d)
    d.global_position = global_position
    d.modulate = color
    await get_tree().create_timer(3.0).timeout
    d.queue_free()
```

### 9.3 使用

- `Jump.enter()`：绿色标记。
- `Jump` 转 `Fall` 前：黄色标记。
- `Fall` 落地时：红色标记。
