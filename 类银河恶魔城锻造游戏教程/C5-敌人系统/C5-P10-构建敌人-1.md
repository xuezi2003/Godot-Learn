# C5-P10-1：构建敌人（上）

> 本集继续扩展敌人系统：为 `ES_Attack` 添加基于曲线的动态移动，并展示已完成的飞行敌人 Demon Chest（恶魔宝箱）的实现要点。


## 1. 攻击状态的曲线移动

### 目标

让敌人在攻击过程中不是匀速移动，而是根据攻击进度动态加速/减速/后退。

### 新增导出变量

```gdscript
@export var move_speed_curve: Curve
```

### 采样逻辑

```gdscript
func physics_update(delta: float) -> void:
    timer += delta
    var sample := 1.0
    if move_speed_curve:
        sample = move_speed_curve.sample(timer / duration)
    enemy.velocity.x = move_speed * sample * blackboard.dir
```

### 曲线示例

| 时间比例 | 速度比例 | 效果 |
|----------|----------|------|
| 0.0 | 0.0 | 攻击开始不动 |
| 0.5 | 1.0 | 攻击中段全速前冲 |
| 0.9 | 0.0 | 攻击结束减速 |

- 曲线 X、Y 范围均为 0–1，便于归一化。
- 可设置 Y 最小值/最大值超过 1，实现更快前冲或反向撤退。


## 2. Demon Chest（恶魔宝箱）

### 特点

- 飞行敌人，可穿越单向平台。
- 状态：飞行、飞行追逐、飞行攻击、眩晕、死亡。
- 使用 `FlyingDecisionEngine`。

### 飞行状态与普通行走状态的区别

| 特性 | 行走 | 飞行 |
|------|------|------|
| 速度维度 | 仅 X | X 与 Y |
| 重力 | 启用 | 关闭 |
| 平台碰撞 | 地面 + 单向平台 | 仅地面 |

### 飞行状态示例

```gdscript
func physics_update(delta: float) -> void:
    enemy.velocity = Vector2(fly_speed * blackboard.dir, 0.0)
```

### 飞行追逐状态

```gdscript
var direction = enemy.global_position.direction_to(target.global_position)
enemy.change_direction(sign(direction.x))
enemy.velocity = direction * chase_speed
```

### 飞行攻击状态

- 在 `physics_update` 中持续重新计算指向玩家的方向（导向攻击）。
- 也可改为 `enter()` 中只计算一次方向（直线冲刺）。
- 使用曲线控制冲刺速度。


## 3. 碰撞遮罩设置

- 飞行敌人关闭单向平台层（Layer 2）的 Mask，使其可自由穿过平台。
- 根节点关闭重力：`gravity_enabled = false`。
