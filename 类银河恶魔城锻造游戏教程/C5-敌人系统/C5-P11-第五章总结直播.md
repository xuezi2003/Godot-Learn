## 1. 受击闪烁效果 `DamageModulate`

### 目标

- 敌人/可破坏物被击中时短暂闪烁，增强打击感。
- 模块化节点，拖到任意有 `Enemy` 或 `DamageArea` 的场景中即可。

### 脚本要点

```gdscript
class_name DamageModulate
extends Node

@export var flash_color: Color = Color(5, 0, 0, 1)   # 高亮红色
var tween: Tween

func _ready() -> void:
    if owner is Enemy:
        owner.hit.connect(_on_hit)
    else:
        for c in owner.get_children():
            if c is DamageArea:
                c.damage_taken.connect(_on_hit)

func _on_hit(_attack_area: AttackArea) -> void:
    if tween and tween.is_valid():
        tween.kill()
    tween = create_tween()
    owner.modulate = flash_color
    tween.tween_property(owner, "modulate", Color.WHITE, 0.5)
```

### 关键细节

- 使用 `Tween` 避免多个闪烁叠加。
- 闪烁颜色可导出，便于不同敌人自定义。
- 对 `Enemy` 连接 `hit`；对普通可破坏物连接 `DamageArea.damage_taken`。


## 2. 启用 HDR 2D 与发光

### 项目设置

- `Project Settings -> Rendering -> Viewport`：启用 `HDR 2D`。

### 世界环境

- 添加 `WorldEnvironment` 节点，创建 `Environment` 资源。
- 背景模式设为 `Canvas`（2D 发光必需）。
- 开启 `Glow`：
  - 强度（Strength）：约 1.2
  - 阈值（HDR Threshold）：约 0.9
  - 可根据场景微调。

### 效果

- 亮度超过白色的颜色会产生光晕。
- 受击闪烁、玩家技能、火焰等会自然发光。
- 不同关卡可保存不同的 `Environment` 资源并动态切换。

### 注意事项

- 网页导出可能颜色略有偏差。
- 需在不同关卡反复测试，避免过曝。


## 3. 重构拾取物基类 `Pickup`

### 动机

- `HealthPickup` 与 `Coin` 有大量重复代码（弹跳物理、玩家检测、音效）。
- 抽象为基类后，新拾取物只需实现 `on_pickup(player)`。

### 基类结构

```gdscript
class_name Pickup
extends CharacterBody2D

@export var bounce_count: int = 8
@export var bounciness: float = 0.5
@export var pickup_audio: AudioStream
@export var bounce_audio: AudioStream

@onready var area: Area2D

func _ready() -> void:
    for c in get_children():
        if c is Area2D:
            area = c
            area.body_entered.connect(_on_body_entered)
            break

func _physics_process(delta: float) -> void:
    if bounce_count > 0:
        velocity += get_gravity() * delta
        var collision = move_and_collide(velocity * delta)
        if collision:
            bounce_count -= 1
            velocity = velocity.bounce(collision.get_normal()) * bounciness
            if bounce_audio:
                AudioManager.play_sound_at_position(bounce_audio, global_position)

func _on_body_entered(body: Node2D) -> void:
    if body is Player:
        area.body_entered.disconnect(_on_body_entered)
        if pickup_audio:
            AudioManager.play_sound_at_position(pickup_audio, global_position)
        on_pickup(body)
        queue_free()

@abstract
func on_pickup(player: Player) -> void:
    pass
```

### 子类示例：Coin

```gdscript
class_name Coin
extends Pickup

@export var value: int = 1

func on_pickup(player: Player) -> void:
    player.gold += value
```

### 子类示例：HealthPickup

```gdscript
class_name HealthPickup
extends Pickup

@export var heal_amount: float = 10.0

func on_pickup(player: Player) -> void:
    player.hp += heal_amount
```


## 4. 金币系统

### 金币资产

- 在 Aseprite 中绘制 6×6 或 8×8 像素硬币。
- 制作旋转动画：正面 → 侧面 → 背面 → 侧面 → 正面。

### 金币场景

```
Coin (Pickup)
├── Sprite2D / AnimatedSprite2D
├── AnimationPlayer
├── CollisionShape2D
└── Area2D
```

### 玩家金币属性

```gdscript
@export var gold: int = 0:
    set(value):
        gold = value
        gold_changed.emit(gold)

signal gold_changed(amount: int)
```

### HUD 显示

- 在 `PlayerHUD` 中添加 `HBoxContainer`。
- 左侧放硬币动画/纹理，右侧放 `Label`（金币数量）。
- 连接 `MessageBus.gold_changed` 更新标签文本。

### 存档

- 在 `SaveData` 中加入 `gold` 字段。
- 保存时写入 `player.gold`，加载时读取。


## 5. 敌人掉落金币

- 在 `LootDropper` 的 `items` 数组中添加 `LootData`。
- 设置掉落物为 `Coin.tscn`。
- 可配置最小/最大数量与掉落概率。
