# C5-P04：敌人设计展示

> 本集是社区敌人设计展示，逐一点评了观众提交的敌人创意，并挑选出部分作品（如 Demon Chest、Corrupted Tree Stump、Tweety、Living Armor、Doom Scribe 等）后续实现进项目。


## 1. 展示目的

- 让观众了解如何把敌人概念转化为可游玩的敌人。
- 分享像素艺术与动画制作的实用反馈。
- 选出部分投稿作为后续章节的实现对象。


## 2. 投稿作品与点评要点

| 投稿者 | 敌人名 | 核心创意 | 反馈重点 |
|--------|--------|----------|----------|
| Harmless Trust | **Demon Chest / 恶魔宝箱** | 拟态怪，待机时像普通宝箱，玩家靠近后苏醒并飞扑攻击 | 建议增加“伪装成宝箱”的待机状态；动画与时机需微调 |
| Alex Honor | **Corrupted Tree Stump / 腐化树桩** | 双形态：治疗树桩 vs 毒蝇伞腐化形态；攻击方式独特 | 风格与游戏关卡高度契合；建议增大体型，增强障碍感 |
| Ash | **Tweety（可爱鬼魂）** | 幽灵跟随玩家并偷取物品 | 浮动/跟随动画出色；死亡溶解效果可再打磨 |
| Karen | **Pixel Art Bandit / 像素强盗** | 大量可替换部件（头发、盔甲、武器） | 变体系统很酷但文件管理复杂；建议统一尺寸后导出精灵表 |
| Myra Doodles | **Giant Worm / 巨虫** | 行走 + 咬合 + 死亡动画 | 高对比度配色易读；死亡淡出可用引擎实现，无需多帧 |
| Chonky Bites | **Golem / 魔像**、**Abomination / 憎恶** | 石头魔像，远程魔法炮 + 近战剑臂 | 建议体型更大以增强存在感 |
| Dr. Dolzvil | **Doom Scribe / 末日书记官** | 固定不动，绘制诅咒符文攻击玩家 | 文档详尽，游戏完成度高；建议提升角色本身对比度 |
| Pupubu | **Bomb Monster / 炸弹怪** | 靠近玩家后自爆 | 爆炸动画建议单独拆出；注意旋转像素艺术产生的瑕疵 |
| Robzon | **Living Armor / 活体盔甲** | 铠甲中游荡的灵魂，攻击动作精美 | 动画流畅，配色简洁；建议在浅色背景前提升对比度 |
| Robzon | **Shadow / 暗影幽灵** | 黑色身体 + 飘逸长发，动作由头发传递 | 最佳像素工艺示范；截图未能体现动画精髓 |
| Lucky Loop | **Nature's Undead / 自然的活死人** | 漂浮头骨，近战 + 远程酸液攻击 | itch.io 页面出色；建议增强轮廓对比度 |
| P Xavier | **Fly Eye / 苍蝇眼** | 飞行俯冲敌人 | 注意旋转产生的像素瑕疵；爆炸动画干净 |
| Witty Username | **Dungeon Rat / 地牢鼠** | 简单老鼠，带帽子变体 | 尺寸合适；建议增强腿部阴影，帽子可随动作弹动 |
| Earthpainting | **Vertecray / 垂直射线** | 从天花板砸下，脊柱可当梯子攀爬 | 最独特的设计理念；需要配合梯子机制 |


## 3. 获奖/特别推荐

| 奖项 | 获奖作品 | 理由 |
|------|----------|------|
| **最佳设计理念** | Vertecray | 把敌人脊柱当梯子使用，玩法独特 |
| **最具备游戏完成度** | Doom Scribe | 精灵表、动画文档、实现思路都很完整 |
| **最佳像素工艺** | Shadow（Robzon） | 头发/布料的动画表现力极强 |
| **最契合本游戏风格** | Corrupted Tree Stump | 与游戏世界美术无缝融合 |
