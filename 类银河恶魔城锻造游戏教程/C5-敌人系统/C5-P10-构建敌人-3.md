# C5-P10-3：构建敌人（下）

> 本集实现社区投稿的 **Doom Scribe（末日书记官）**：一个固定不动的远程/法术型敌人，玩家进入范围后开始施法，若在法术完成前击杀敌人则法术消散。


## 1. Doom Scribe 特点

- 不移动、不受重力影响。
- 玩家进入圆形探测范围后开始施法。
- 施法持续一段时间，然后在玩家头顶生成诅咒法术并造成伤害。
- 若敌人在法术命中前死亡，诅咒自动消失。


## 2. 场景准备

- 复制 Demon Chest 场景，替换精灵为 `doom_scribe`。
- 精灵表：`6×5`，共 30 帧。
- 关闭重力。
- 删除 `AttackArea`，改用独立法术场景造成伤害。


## 3. 动画设置

| 动画名 | 帧范围 | 循环 | 说明 |
|--------|--------|------|------|
| `idle` | 0–5 | 是 | 上下浮动，循环 |
| `attack` | 7–14 | 是 | 在卷轴上书写/施法 |
| `stun` | 15 | 否 | 单帧受击 |
| `death` | 16 往后 | 否 | 融化倒地 |


## 4. 法术场景 `doom_scribe_curse`

### 节点结构

```
DoomScribeCurse (Node2D)
├── Sprite2D
├── AnimationPlayer         # 自动播放，无循环
└── AttackArea              # 伤害区域，默认禁用
    └── CollisionShape2D
```

### 脚本要点

```gdscript
class_name DoomScribeCurse
extends Node2D

@export var damage: float = 10.0
var enemy: Enemy

func _enter_tree() -> void:
    await get_tree().process_frame
    var player = get_tree().get_first_node_in_group("player")
    if player:
        reparent(player)
        position = Vector2(0, -48)
    if enemy and enemy.has_method("set_enemy"):
        enemy.killed.connect(queue_free)

func hurt_player() -> void:
    $AttackArea.activate(0.2)

func _on_animation_finished(_anim_name: String) -> void:
    queue_free()
```


## 5. ES_AttackSpell（法术攻击状态）

- 继承 `ES_Attack`。
- 新增导出变量：
  ```gdscript
  @export var cast_duration: float = 1.0
  @export_file("*.tscn") var spell_scene: String
  ```
- `enter()`：
  - 调用 `super.enter()` 播放攻击动画。
  - `duration = cast_duration`。
- `physics_update()`：
  - 计时结束即视为施法完成，实例化法术场景。
- 实例化后：
  - 调用 `spell.set_enemy(enemy)`（若存在），使敌人死亡时法术消散。
  - 作为敌人的同级节点加入场景。


## 6. 法术命中

- 在诅咒动画的关键帧调用 `hurt_player()`。
- 激活 `AttackArea` 一瞬间，对玩家造成伤害。
- `AttackArea` 的 Mask 设为玩家层。


## 7. 决策引擎

- 可直接复用 `DecisionEngine_BasicAttack`。
- 把 `walk_state` 分配为 `idle` 动画的状态（移动速度为 0）。
- 把 `attack_state` 分配为 `ES_AttackSpell`。


## 8. 关键机制总结

- 敌人和法术通过信号解耦：法术监听敌人的 `killed` 信号。
- 法术位置跟随玩家（`reparent`），保证命中判定准确。
- 击杀敌人即可避免伤害，增加策略性。


## 9. 扩展方向

- 把 `DoomScribeCurse` 泛化为通用 `Projectile`/`Spell` 基类。
- 为法术添加目标参数，自动追踪或直线飞行。
- 添加施法预警（符文变红）提示玩家尽快打断。
