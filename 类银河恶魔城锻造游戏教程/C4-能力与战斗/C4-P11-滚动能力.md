# C4-P11：滚动能力（变形球）

> 本集实现一个”变形球”状态，让玩家缩小体型穿过狭窄通道，同时处理碰撞形状切换、射线检测、平台跳下以及与交互系统的冲突避免。

## 1. 核心设计

- 独立状态 `BallState`，切换到更小碰撞体。
- 球形态下可左右滚动、跳跃、跳下单向平台。
- 只有在上方与下方都有足够空间时才能解除变形，防止卡住。
- 通过 `morph` 布尔值与 `can_morph()` 控制是否拥有该能力。

## 2. 资源准备

- 导入变形进入/退出音效。
- 玩家精灵表中包含球滚动动画帧（第 44–46 帧）。
- 在关卡中搭建玩家无法步行通过的狭窄通道。

## 3. 球滚动动画

- 新建动画 `ball`，时长 `0.225` 秒，3 帧，循环播放。
- 玩家精灵的 `flip_h` 会自动处理左右方向。

## 4. 创建 `BallState`

### 变量

```gdscript
class_name PlayerStateBall
extends PlayerState

@export var ball_jump_speed: float = 400.0

var on_floor: bool = true
var direction: float = 1.0

@onready var collision_shape: CollisionShape2D = %CollisionShape_Stand
@onready var damage_shape: CollisionShape2D = %DA_Stand
@onready var ray_up: RayCast2D = %BallRay_Up
@onready var ray_down: RayCast2D = %BallRay_Down
@onready var morph_in_audio: AudioStream = preload("res://.../morph_in.wav")
@onready var morph_out_audio: AudioStream = preload("res://.../morph_out.wav")
```

### `enter`

- 播放 `ball` 动画。
- 将胶囊碰撞形状缩小为球形：
  - 半径 `11`，高度 `22`（直径）。
  - 将 `collision_shape.position.y` 与 `damage_shape.position.y` 都设为 `-11`。
- 给玩家一点向上的 Y 速度反弹，增加变形活力感。
- 播放 `morph_in_audio`。

```gdscript
func enter() -> void:
    player.animation_player.play("ball")
    var shape := collision_shape.shape as CapsuleShape2D
    shape.radius = 11.0
    shape.height = 22.0
    collision_shape.position.y = -11.0
    damage_shape.position.y = -11.0
    player.velocity.y -= 100.0
    Audio.play_spatial_sound(morph_in_audio, player.global_position)
```

### `_physics_process`

- 水平移动：`player.velocity.x = player.direction.x * player.move_speed`。
- 检测是否接触地面，用于后续跳跃与落地反馈。

### `_process`（动画速度缩放）

```gdscript
func _process(_delta: float) -> void:
    if player.direction.x == 0.0:
        player.animation_player.speed_scale = 0.0
    else:
        player.animation_player.speed_scale = 1.0
```

- 停止移动时暂停滚动动画，移动时恢复。

### 输入处理

```gdscript
func _input(event: InputEvent) -> PlayerState:
    if event.is_action_pressed("morph") and _can_stand():
        Audio.play_spatial_sound(morph_out_audio, player.global_position)
        return player.idle_state if player.is_on_floor() else player.fall_state

    if event.is_action_pressed("jump"):
        if player.is_on_floor():
            if player.direction.y > 0.0 and player.one_way_cast.is_colliding():
                player.position.y += 4.0
                return null
            player.velocity.y = -ball_jump_speed
            jump_audio.play()
            VisualEffects.spawn_jump_dust(player.global_position)
    return null
```

### 检测能否站立 `_can_stand()`

```gdscript
func _can_stand() -> bool:
    ray_up.enabled = true
    ray_down.enabled = true
    ray_up.force_raycast_update()
    ray_down.force_raycast_update()
    var blocked := ray_up.is_colliding() and ray_down.is_colliding()
    ray_up.enabled = false
    ray_down.enabled = false
    return not blocked
```

- 两条射线都碰撞说明空间不足，不能解除变形。
- 射线配置：
  - `RayCast2D` 目标位置向上/向下约 43 像素，略微偏移避免贴地误检。
  - 掩码只与地形碰撞，开启 `collide_with_bodies`。

### `exit`

- 恢复胶囊形状默认半径 `8`、高度 `46`、Y 位置 `-23`。
- 恢复 `animation_player.speed_scale = 1.0`。
- 播放 `morph_out_audio`。

## 5. 状态连接

在 `Idle`、`Run`、`Jump`、`Fall`、`Crouch`、`Attack`、`Dash` 的输入处理中加入：

```gdscript
if event.is_action_pressed("morph") and player.can_morph():
    return ball_state
```

## 6. 玩家脚本：`can_morph()`

```gdscript
@export var morph: bool = false
var can_interact: bool = false

func can_morph() -> bool:
    if not morph:
        return false
    if can_interact:
        return false
    return true
```

### 交互检测

- 连接 `MessageBus.input_prompt_changed` 信号。
- 当提示为 `"interact"` 时设置 `can_interact = true`，否则为 `false`。
- 这样站在存档点/门前按互动键时不会误触发变形。

## 7. 落地与跳跃反馈

- 球形态跳跃时播放跳跃音效与跳跃尘埃。
- 落地时播放落地音效与落地尘埃。
- 通过 `on_floor` 变量在 `_physics_process` 中检测“离开地面”/“接触地面”的转换。

## 8. 关卡碰撞微调

- 给狭窄通道顶部的瓦片添加少量物理碰撞圆角，防止玩家主体意外滑入缝隙。
- 调整通道高度或球体尺寸，使球刚好能通过但正常体型不能。
