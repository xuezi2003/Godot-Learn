# C4-P07：玩家死亡与游戏结束

> 本集在眩晕状态基础上实现死亡状态，并在 `PlayerHUD` 中构建一个淡入式的游戏结束界面，支持”加载上次存档”与”退出至标题”。

## 1. 准备工作

- 提高尖刺假人伤害（例如从 1 提升到 10），让死亡更容易测试。
- 死亡动画帧已内置在玩家精灵表第 47–60 帧。
- 导入死亡音效。

## 2. 游戏结束 UI

### 场景结构（在 `PlayerHUD` 内）

- `GameOver`（`Control`，全屏尺寸，以唯一名称访问）
  - `ColorRect`：黑色半透明背景幕布。
  - `Label`：红色文字“GAME OVER”。
  - `HBoxContainer`
    - `LoadButton`：“加载上次存档”
    - `QuitButton`：“退出至标题”

### `PlayerHUD.gd` 脚本

#### 默认隐藏

```gdscript
@onready var game_over: Control = %GameOver
@onready var load_button: Button = %LoadButton
@onready var quit_button: Button = %QuitButton

func _ready() -> void:
    game_over.visible = false
    load_button.pressed.connect(_on_load_pressed)
    quit_button.pressed.connect(_on_quit_pressed)
```

#### 显示游戏结束

```gdscript
func show_game_over() -> void:
    load_button.visible = false
    quit_button.visible = false
    game_over.modulate.a = 0.0
    game_over.visible = true

    var tween := create_tween()
    tween.tween_property(game_over, "modulate", Color.WHITE, 3.0)
    await tween.finished

    load_button.visible = true
    quit_button.visible = true
    load_button.grab_focus()
```

- 先隐藏按钮，淡入背景，再显示按钮并聚焦。

#### 清除游戏结束

```gdscript
func clear_game_over() -> void:
    game_over.visible = false
    var player = get_tree().get_first_node_in_group("player")
    if player:
        player.queue_free()
```

- 在“加载”与“退出”按钮处理中都会调用。
- 移除旧玩家，避免加载后继承死亡状态或等级异常。

#### 按钮逻辑

```gdscript
func _on_load_pressed() -> void:
    SaveManager.load_game(SaveManager.current_slot)

func _on_quit_pressed() -> void:
    SceneManager.transition_to_scene(title_screen, "", Vector2.ZERO, TransitionType.UP)
```

- 注意 `SceneManager` 的 `scene_entered` 信号触发时调用 `clear_game_over`，避免游戏结束画面残留在过渡后。

## 3. 死亡状态 `DeathState`

### 脚本要点

```gdscript
class_name PlayerStateDeath
extends PlayerState

@onready var death_audio: AudioStream = preload("res://.../player_death.wav")

func enter() -> void:
    player.velocity.x = 0.0
    player.animation_player.play("death")
    Audio.play_spatial_sound(death_audio, player.global_position)
    Audio.play_music("")  # 淡出音乐

    await player.animation_player.animation_finished
    PlayerHud.show_game_over()
```

- 进入时停止水平移动。
- 播放死亡动画与音效，淡出音乐。
- 动画结束后调用 HUD 显示游戏结束。

### 从眩晕状态进入死亡

在 `HurtState` 的计时结束分支中：

```gdscript
if player.hp <= 0:
    next_state = player.death_state
else:
    next_state = player.idle_state if player.is_on_floor() else player.fall_state
```

这样玩家在受击短暂眩晕后再死亡，而不是立即死亡。

### 防止死亡后继续受伤

在 `HurtState._on_damage_taken` 与玩家受伤处理中加入检查：

```gdscript
if player.current_state == player.death_state:
    return
```

## 4. 测试

- 用测试按钮直接触发 `show_game_over()` 验证 UI 动画。
- 被尖刺假人多次击中后进入死亡状态。
- 选择“加载上次存档”重生并恢复满血。
- 选择“退出至标题”返回标题画面。

## 5. 注意点

- `SceneManager` 的信号触发时机可能因实现而异；若 `load_scene_started` 信号在订阅前已发出，可改用 `scene_entered` 来清理 UI。
- 加载新场景前移除旧玩家，能避免死亡状态、等级等数据污染新实例。
