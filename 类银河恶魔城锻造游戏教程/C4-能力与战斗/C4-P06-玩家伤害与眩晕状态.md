# C4-P06：玩家伤害与眩晕状态

> 本集构建一个始终激活的”危险区域”，并给玩家添加受击判定与眩晕状态，使玩家能真正受到伤害、进入无敌帧，并通过 HUD 看到生命值变化。

## 1. 新节点：危险区域 `HazardArea`

### 设计目的

- 普通 `AttackArea` 只在攻击时短暂激活。
- `HazardArea` 继承 `AttackArea`，但**始终处于监控状态**，适合尖刺、敌人身体等环境危害。

### 脚本

```gdscript
class_name HazardArea
extends AttackArea

func _ready() -> void:
    super._ready()
    monitoring = true
    monitorable = true
```

- 调用 `super._ready()` 保留父类逻辑。

### 尖刺假人

1. 复制 `TestDummy` 场景为 `SpikeDummy`。
2. 将身体精灵调整为带刺外观。
3. 在身体精灵下添加 `HazardArea` + 矩形碰撞形状。
4.  hazard 区域会随身体摇摆而旋转，扩大危险范围。
5. 增加假人的摇摆幅度/速度，使其更容易伤到玩家。

## 2. 玩家伤害区域

### 场景结构

- 在玩家根节点下新增 `DamageArea`（受击判定/伤害区域）。
- 复制站立与蹲伏的碰撞形状作为 `DA_Stand` 与 `DA_Crouch` 子节点。
  - 由于是复制同一资源，形状改动会同步。
- 默认启用 `DA_Stand`，禁用 `DA_Crouch`。

### 碰撞层

- `DamageArea` 放在**玩家层**。
- `HazardArea` 的碰撞掩码检查玩家层。

### 蹲伏切换

- 在 `CrouchState.enter()` 中启用 `DA_Crouch`，禁用 `DA_Stand`。
- 在 `CrouchState.exit()` 中恢复。
- 这样玩家下蹲时可躲避高处判定。

## 3. 玩家脚本处理受伤

### 变量与信号

```gdscript
signal damage_taken

@onready var damage_area: DamageArea = %DamageArea
```

### `_ready`

```gdscript
func _ready() -> void:
    damage_area.damage_taken.connect(_on_damage_taken)
```

### `_on_damage_taken`

```gdscript
func _on_damage_taken(attack_area: AttackArea) -> void:
    hp -= attack_area.damage
    damage_taken.emit()
```

- `hp` 使用 setter，会自动限制在 `[0, max_hp]` 并发出 `player_health_changed` 信号，HUD 血条随之更新。
- 在 `_ready()` 中设置 `hp = max_hp`，确保初始血条尺寸正确。

## 4. 眩晕状态 `HurtState`

### 状态初始化

- `HurtState` 不是由输入触发，而是由外部事件触发。
- 在 `init()` 中连接玩家 `DamageArea` 的 `damage_taken` 信号：

```gdscript
func init() -> void:
    player.damage_area.damage_taken.connect(_on_damage_taken)

func _on_damage_taken(attack_area: AttackArea) -> void:
    if player.current_state == player.death_state:
        return
    var dir := 1.0 if attack_area.global_position.x < player.global_position.x else -1.0
    _hit_direction = dir
    player.change_state(self)
```

### 变量

```gdscript
@export var knockback_speed: float = 100.0
@export var invincible_duration: float = 0.5

var _timer: float = 0.0
var _hit_direction: float = 1.0
```

### `enter`

- 播放受伤动画（单帧或自定义）。
- 开启无敌：`damage_area.start_invincible(invincible_duration)`。
- 播放受伤音效（通过 `AudioStreamPlayer2D` 直接播放，作为玩家局部音效的示例）。
  - > 若使用直接挂在玩家身上的 `AudioStreamPlayer2D`，记得在检查器中将其 **Bus** 设为 **SFX**；全局音频系统会自动处理，但局部播放器需要手动指定总线。
- 触发屏幕震动：`VisualEffects.shake_camera(2.0)`。

### `_physics_process`

- 计时：`timer -= delta`。
- 击退：`player.velocity.x = _hit_direction * knockback_speed`。
- 当 `timer <= 0`：
  - 若 `player.hp <= 0`：返回 `DeathState`（为下集铺垫）。
  - 否则返回 `IdleState` 或 `FallState`。

### 无敌效果

- `DamageArea.start_invincible()` 会在持续时间内禁用受伤检测，避免被连续击中硬直。
- 若需要视觉反馈，可添加闪烁或半透明效果。

## 5. 调试与手感
