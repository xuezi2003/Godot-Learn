# C4-P09：冲刺能力

> 本集创建一个独立的 `DashState`，实现冲刺时的快速位移、无敌帧、颜色闪烁与残影拖尾效果，并限制空中只能冲刺一次。

## 1. 核心设计

- 作为**独立状态**，保持状态机职责清晰。
- 冲刺期间无敌，可用于穿越敌人和危险区域。
- 地面可连续冲刺；空中只能冲刺一次，落地后重置。
- 通过 `dash_count` 与 `can_dash()` 函数控制可用性。

## 2. 玩家脚本更新

```gdscript
@export var dash: bool = false
var dash_count: int = 0

func can_dash() -> bool:
    if not dash:
        return false
    if dash_count > 0:
        return false
    return true
```

## 3. `DashState.gd`

### 导出变量

```gdscript
@export var duration: float = 0.25
@export var speed: float = 300.0
@export var effect_delay: float = 0.03
```

### 局部变量

```gdscript
var direction: float = 1.0
var time: float = 0.0
var effect_time: float = 0.0
```

### `enter`

```gdscript
func enter() -> void:
    time = duration
    effect_time = 0.0
    player.dash_count += 1
    player.velocity.y = 0.0
    player.gravity_multiplier = 0.0

    direction = _get_direction()
    player.damage_area.start_invincible(duration)
    Audio.play_spatial_sound(dash_audio, player.global_position)
    player.animation_player.play("dash")
    player.sprite.tween_color()  # 绿色闪烁
```

### `_get_direction`

```gdscript
func _get_direction() -> float:
    return -1.0 if player.sprite.flip_h else 1.0
```

### `_physics_process`

```gdscript
func _physics_process(delta: float) -> void:
    time -= delta
    # 速度从 2*speed 渐变到 speed，营造爆发感
    player.velocity.x = (speed + speed * (time / duration)) * direction

    if time <= 0.0:
        next_state = player.idle_state if player.is_on_floor() else player.fall_state
```

### `exit`

- 恢复 `player.gravity_multiplier = 1.0`。
- 注意：不在这里重置 `dash_count`，否则空中冲刺无意义。

## 4. 状态连接

在 `Idle`、`Run`、`Crouch`、`Jump`、`Fall`、`Attack`、`Dash` 的输入处理中加入：

```gdscript
if event.is_action_pressed("dash") and player.can_dash():
    return dash_state
```

- 冲刺可取消攻击，增加连招可能性。

## 5. 冲刺计数重置

- 在 `IdleState.enter()` 中重置 `player.dash_count = 0`。
- 由于大多数状态最终回到 `Idle`，落地时自然恢复冲刺。

## 6. 土狼时间修正

- 若从 `DashState` 进入 `FallState`，不应触发土狼时间跳跃。
- 在 `FallState.enter()` 中加入：

```gdscript
if previous_state in [jump_state, attack_state, dash_state]:
    coyote_timer = 0.0
```

## 7. 视觉效果

### 冲刺动画

- 复制待机动画为 `dash`，单帧姿势（第 29 帧），循环播放。

### 玩家精灵颜色补间

新建 `PlayerSprite.gd` 并附加到玩家精灵：

```gdscript
class_name PlayerSprite
extends Sprite2D

var tween: Tween

func tween_color(color: Color = Color(0.0, 1.5, 0.95), duration: float = 0.3) -> void:
    if tween != null:
        tween.kill()
    modulate = color
    tween = create_tween()
    tween.tween_property(self, "modulate", Color.WHITE, duration)
```

### 残影拖尾

```gdscript
func ghost() -> void:
    var effect := Node2D.new()
    player.get_parent().add_child(effect)
    effect.global_position = player.global_position
    effect.z_index = 1

    var copy := duplicate() as Sprite2D
    effect.add_child(copy)
    effect.modulate = Color(0.0, 1.5, 0.95, 0.75)

    var t := create_tween().set_ease(Tween.EASE_OUT)
    t.tween_property(effect, "modulate:a", 0.0, 0.2)
    t.finished.connect(effect.queue_free)
```

- 在 `DashState._process` 中按 `effect_delay` 间隔调用 `player.sprite.ghost()`。
- 通过 `get_parent().move_child(effect, 0)` 让残影位于玩家后方。

## 8. 结果

- 冲刺手感迅捷有力，可衔接跳跃、二段跳、攻击。
- 无敌帧让冲刺具备防御价值。
- 绿色闪光 + 残影拖尾让能力极具辨识度。
