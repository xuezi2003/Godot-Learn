# C2-P2：瓦片集资源创建

> 在 Aseprite 中绘制 32×32 像素瓦片，分别制作手动拼贴瓦片与自动地形瓦片，并导出为 Godot 可用的精灵表。

## 1. 文档设置

- 瓦片尺寸：**32×32 像素**。
- 推荐画布尺寸为 32 的整倍数且尽量为 2 的幂，例如 **1024×512**。
- 在 Aseprite 中将网格改为 **32×32**，确保所有内容对齐网格。

## 2. 两种瓦片制作方式

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 手动拼贴瓦片 | 每个部件独立绘制，可表现丰富细节与不规则边缘 | 复杂平台、悬浮岛屿、装饰性地面 |
| 自动地形瓦片 | 按模板一次性绘制所有相邻组合，绘制时自动连接 | 大面积地形、土/石填充区域 |

## 3. 手动拼贴瓦片要点

- 从深色底色开始，逐步向浅色过渡。
- 为平台顶部保留草皮或材质表面，两侧用岩石/砖块填充。
- 添加高光、阴影和裂纹，增加体积感。
- 确保各部件能无缝平铺；可用复制、翻转、粘贴快速生成对称部件。
- 绘制角落部件时，把相邻边拼在一起检查过渡是否自然。

## 4. 自动地形模板

- 使用 47 块组合模板（Blob 模板），覆盖孤立块、四边连接、边角等所有组合。
- 模板中粉色/示例块仅用于展示连接逻辑，绘制时用自己的纹理替换。
- 绘制顺序建议：先画孤立块，再画各边连接、角落、最后被包围的中心块。
- 模板文件已包含在第二章素材包中，可直接在此基础上绘制。

## 5. Aseprite 快捷键与技巧

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + B` | 将当前选区转换为画笔，方便重复绘制同一图案 |
| `Shift + H` | 水平翻转选区或当前帧 |
| `右键` | 使用当前橡皮擦设置擦除（继承画笔尺寸/模式） |
| `Alt + 点击` | 吸取颜色；在两种颜色间点击可取中间色 |
| `Ctrl + '`（引号） | 切换网格显示 |
| `Ctrl + T` | 自由变换选区/帧，常用于制作动画中间帧 |

### 像素完美模式

- 开启后可避免画出孤立像素点，适合快速绘制硬边像素形状。
- 关闭后更容易画出锯齿状、自然的岩石纹理，作者通常在打底稿时关闭。

### 网格颜色

- 编辑 → 偏好设置 → 网格，可修改网格颜色以增强对比度（如蓝绿色或紫色）。
- 当瓦片颜色与默认网格接近时，换颜色能大幅减少眼睛疲劳。

### 动画标签与 Ping-Pong

1. 选中连续帧 → 右键 → 新建标签，命名如 `Leaf`。
2. 标签属性中将动画方向设为 **Ping-Pong**，可让动画从首帧播到末帧再倒回，减少手动绘制帧数。
3. 用洋葱皮（Onion Skin）查看前后帧，辅助对齐中间帧。

### 导出精灵表

- 文件 → 导出 → 导出精灵表（`Ctrl + E`）。
- 排列方式：**按行（By Rows）**。
- 固定列数：按需求设置（如 16），确保是单帧宽度的整数倍。
- 必须勾选 **输出到文件**，否则导出按钮会灰显无法保存。
- 作者习惯将所有道具/动画合并到一个精灵表中，减少 Godot 加载的文件数量。

## 6. 在 Godot 中快速验证

- 将完成的精灵表导出为 PNG，导入 Godot 的 TileSet。
- 用 TileMapLayer 绘制，检查平铺接缝与自动地形连接是否正确。
- 出现问题时回到 Aseprite 修改，重新导出后 Godot 会自动更新。

## 7. 道具与动画

- 道具建议单独建立精灵文件，方便制作动画。
- 示例：制作一株随风摇摆的植物。
  - 绘制第 0 帧与最后一帧作为关键帧。
  - 在中间插入过渡帧，使用洋葱皮辅助对齐。
  - 将动画类型设为 **Ping-Pong**，减少手动绘制帧数。
- 导出时选择按行排列，便于 Godot 切片。

## 8. 纹理图集的性能优势

- 将多种瓦片、道具合并到一张大图，减少文件加载次数与显存占用。
- 在 3D 中称为 Texture Atlas，2D 瓦片集同理。

## 9. 课后任务

- 用 Aseprite 绘制自己的 32×32 瓦片集。
- 至少包含：地面、平台、角落、斜坡、背景墙、几种装饰植物。
- 尝试制作一个自动地形瓦片组。
- 从 Itch.io 素材包中可下载作者已完成版本作为参考。
