# C2-P7：动画道具

> 使用 AnimatedSprite2D 制作植物、火把等可复用动画道具，并简要介绍瓦片动画替代方案与性能监控工具。

## 1. 瓦片动画（快速但不灵活的方案）

- 在 TileSet 中选中一个瓦片，进入“选择”选项卡。
- 设置动画列数（如 4），勾选 **Random Start Time**。
- 将四帧加入动画条，即可在 TileMapLayer 中绘制动画瓦片。
- 局限：不支持 Ping-Pong、难以逐帧精细控制、位置受网格约束。
- 适合一次性动画或不需要精细控制的装饰。

## 2. AnimatedSprite2D 方案

### 植物道具

1. 在场景中创建 **AnimatedSprite2D** 节点。
2. 在 SpriteFrames 资源中点击“从精灵图集添加帧”。
3. 选择瓦片集纹理，设置切片尺寸为 **32×32**。
4. 选中植物的 4 帧，点击添加。
5. 为实现 Ping-Pong 摇摆效果，复制第 1、2 帧并粘贴到末尾：
   - 帧顺序：`0, 1, 2, 3, 2, 1`
6. 调整 **FPS**（如 6）和 **Speed Scale**，可让不同实例播放速度不同。
7. 设置 **Offset Y = -16**，让植物从原点向上生长。
8. 勾选 **Autoplay on Load**。
9. 右键保存为分支场景到 `props/plant.tscn`。

### 火把道具

1. 新建场景，根节点同样使用 **AnimatedSprite2D**。
2. 火把占用 2 个瓦片高度，导入时垂直切片数设为 8（32×64）。
3. 选择 4~5 帧火焰动画，调整 FPS 到 12 左右。
4. 设置 **Offset Y = -32**。
5. 保存为 `props/torch.tscn`。

## 3. 在关卡中摆放

- 创建一个空的 `Props` 节点作为分组容器。
- 将道具实例拖入，可自由移动、翻转、旋转。
- 通过调整每个实例的 **Speed Scale**，让同类道具动画不完全同步，显得更自然。
- 部分道具可放到背景中并调制为偏蓝/暗色调，增加层次。

## 4. 性能监控

- 打开调试器中的 **Monitor** 面板可查看：
  - **VRAM**：纹理占用（如森林瓦片集大小）。
  - **FPS / Frame Time**：当前帧率与每帧耗时。
  - **Physics Time / Process Time**：物理与脚本处理时间。
- **Visual Profiler** 可按函数查看耗时，帮助定位性能瓶颈。
- 动画精灵复用同一张纹理图集，显存占用很低，适合大量摆放。

## 5. 课后任务

- 为每个场景添加植物、火把等动画道具。
- 尝试让部分道具放在背景层、部分放在前景层。
- 使用性能监控工具观察添加道具前后的帧时间与显存变化。
