# C2-P1：世界灵感

> 通过分析参考游戏、建立情绪板与绘制简易区域地图，为本项目确立像素美术方向与世界观框架。

## 1. 参考游戏分析

### Hollow Knight / Silksong

- 视觉风格偏向**卡通化高清 2D**：线条流畅、形状简洁、阴影较少。
- 使用**视差滚动**、粒子效果（雨滴、尘土）营造深度与氛围。
- 每个区域有统一的**近似色配色方案**（如入门区域整体偏绿）。
- 平台多为硬朗锐角；本教程不会完全照搬，但借鉴其氛围控制方法。

### 恶魔城：月下夜想曲

- **高清像素艺术**典范：色彩层次丰富、背景特效华丽、角色动画细腻。
- 角色动作（如跳跃拖尾）极具参考价值。
- 平台边缘有纵深感设计（如角色可“坐”在平台边缘）。
- 内部区域纵深较弱，但整体色彩与细节值得学习。

### Owlboy

- **高清像素艺术 + 丰富细节**：平台边缘有过渡、角色下方有投影。
- 环境充满动态元素：飞虫、植物摇曳、地面草叶摆动。
- 平台边缘用深色渐变过渡到近似黑色，清晰划分可行走区域。
- 仅用少量视差层即可营造强烈纵深感。

## 2. 情绪板（Mood Board）

- 使用 **Affinity / Photoshop** 等软件创建画板，收集参考图、纹理与色彩样本。
- 按区域分类：森林、城堡、地牢等，分别建立独立画板。
- 从参考图中采样主色、高光、阴影，形成受限制的调色板（如 3~5 种主色 + 高光/阴影色）。

### 实践步骤

1. 新建画布（常用屏幕分辨率即可）。
2. 从网上或截图中收集符合氛围的参考图并粘贴到画板。
3. 删除不符合需求的图片，逐步筛选。
4. 用取色工具提取关键色，整理成调色板。
5. 为角色、敌人、Boss 等建立独立参考页。

## 3. 简易世界地图

- 用简单矩形表示每个“屏幕/摄像机视角”大小的关卡片段。
- 用不同颜色区分生态：橙色代表地牢/地下墓穴，紫色代表城堡，绿色代表存档点房间。
- 用 `B` 标注 Boss 房，用绿色方块标注存档点。
- 地图不是铁律，后续会随开发调整；重点是提前规划区域连接方式。

## 4. 课后任务

- 为自己游戏的每个场景搭建情绪板。
- 收集配色、平台样式、道具风格的参考图。
- 在纸上或数字工具中绘制简易区域连接地图。
