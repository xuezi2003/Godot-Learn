# C2-P0：世界构建基础章节概述

> 本集概述第二章“世界构建基础”的学习目标、设计思路与后续各集安排。

## 1. 本章定位

- 第一章完成了可操作的角色原型，第二章开始转向**世界/关卡构建**。
- 类银河恶魔城游戏的核心在于**相互关联的世界**：探索、惊喜与神秘感。
- 本章同时覆盖**设计原则**（氛围、美术方向）与**技术实现**（瓦片地图、关卡切换等）。

## 2. 章节安排

| 集数 | 主题 | 要点 |
|------|------|------|
| P1 | 世界灵感 | 情绪板、参考游戏、概念美术、简易地图 |
| P2 | 瓦片集资源创建 | 在 Aseprite 中绘制 32×32 瓦片、自动地形模板 |
| P3 | 瓦片集与瓦片地图图层 | Godot TileSet / TileMapLayer、碰撞层、地形系统 |
| P4 | 关卡边界与摄像机 | Camera2D、LevelBounds 节点、编辑器预览 |
| P5 | 关卡场景切换系统 | 关卡过渡门、SceneManager、淡入淡出切换 |
| P6 | 视差图层与大气景深 | Parallax2D、多层背景/前景、色彩调制 |
| P7 | 动画道具 | AnimatedSprite2D、植物/火把、可复用道具场景 |
| P9a / P9b | 2D 雾效与粒子 | 噪声纹理雾气、GPU 粒子火花与光点 |
| P10 | 玩家生成与关卡构建指南 | PlayerSpawn 节点、关卡原型设计建议 |

## 3. 核心建议

- 关卡美术不要过早精细化：先确保功能与布局合理，再添加视觉润色，避免返工。
- 世界氛围、色彩方案与光照会显著影响玩家体验。
- 所有资源与图标可在作者的 Itch.io 页面获取。
