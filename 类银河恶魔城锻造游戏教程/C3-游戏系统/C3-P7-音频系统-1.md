# C3-P7：音频系统 - 1

> 本集开始构建全局音频系统，包括音乐和 UI 音频播放、音频总线布局、跨场景音乐切换，以及为 UI 按钮批量添加音效的便捷方法。

## 1. 资源导入

### 1.1 文件夹结构

- 在 `Globals/` 下新建 `Audio/` 文件夹。
- 在 `Audio/` 下新建 `UIAudio/` 子文件夹。
- 在项目根目录新建 `Music/` 文件夹。

### 1.2 导入文件

- **UI 音效**：将带 `UI_` 前缀的 WAV 文件复制到 `Audio/UIAudio/`。
  - `UI_Focus.wav`
  - `UI_Select.wav`
  - `UI_Cancel.wav`
  - `UI_Success.wav`
  - `UI_Error.wav`
- **音乐**：将 OGG 音乐文件复制到 `Music/`。
  - `Title.ogg`
  - `Forest.ogg`
  - `Dungeon.ogg`
- **图标**：确保 `MusicAutoTrigger` 图标在项目根图标文件夹中。
- **门音效**：确保 `DoorBash.wav`、`DoorSwitch.wav` 已在 `Common/DoorAndSwitch/` 中。

### 1.3 音频导入设置

- 选中音乐 OGG 文件，点击 **导入 → 循环 → 开**，然后重新导入。
- WAV 文件默认从文件检测循环模式，无需循环的 UI 音效保持默认即可。
- 可启用 `Normalize` 调整音量峰值，但本项目中音频已相对平衡。

## 2. 音频场景搭建

### 2.1 创建全局音频场景

- 在 `Globals/Audio/` 中新建 `Audio.tscn`。
- 根节点类型改为 **Node**（不需要 2D/3D 位置属性）。
- 命名为 `Audio`。

### 2.2 节点结构

```
Audio (Node)
├── MusicPlayer1 (AudioStreamPlayer)
├── MusicPlayer2 (AudioStreamPlayer)
└── UIAudioPlayer (AudioStreamPlayer)
```

- `MusicPlayer1` / `MusicPlayer2`：用于交叉淡入淡出切换音乐。
- `UIAudioPlayer`：使用 `AudioStreamPolyphonic` 支持多声道同时播放 UI 音效。

### 2.3 自动加载

- 进入 **项目设置 → 自动加载**，添加 `Audio.tscn`，命名为 `Audio`。
- 为 `Audio` 根节点附加脚本 `audio.gd`。

## 3. 音频总线布局

### 3.1 打开音频总线

- 底部面板选择 **音频**。
- 默认只有 `Master` 总线。

### 3.2 添加总线

添加以下总线：

| 总线名 | 用途 | 路由 |
|---|---|---|
| `Master` | 主输出 | 扬声器 |
| `EffectPass` | 添加混响等效果 | Master |
| `Music` | 音乐通道 | EffectPass |
| `SFX` | 游戏音效 | EffectPass |
| `UI` | UI 音效 | Master |

### 3.3 配置播放器总线

- `MusicPlayer1` 与 `MusicPlayer2`：`Bus = Music`
- `UIAudioPlayer`：`Bus = UI`
- 后续空间音效将通过代码设置 `Bus = SFX`

### 3.4 添加混响效果

- 在 `EffectPass` 总线上添加 **Reverb** 效果。
- 后续可通过脚本动态启用/禁用混响并调整房间大小等参数。

## 4. UI 音频系统

### 4.1 导出 UI 音效变量

```gdscript
@export var ui_focus_audio: AudioStream
@export var ui_select_audio: AudioStream
@export var ui_cancel_audio: AudioStream
@export var ui_success_audio: AudioStream
@export var ui_error_audio: AudioStream
```

### 4.2 多声道音频播放器

- 将 `UIAudioPlayer` 的 `Stream` 设为 `AudioStreamPolyphonic`。
- 在 `_ready()` 中获取流播放引用：

```gdscript
var ui_audio_stream_playback: AudioStreamPlaybackPolyphonic

func _ready() -> void:
    ui_audio_player.play()
    ui_audio_stream_playback = ui_audio_player.get_stream_playback()
```

### 4.3 播放 UI 音频

```gdscript
func play_ui_audio(stream: AudioStream) -> void:
    if ui_audio_stream_playback:
        ui_audio_stream_playback.play_stream(stream)
```

### 4.4 便捷函数

```gdscript
func ui_focus_changed() -> void:
    play_ui_audio(ui_focus_audio)

func ui_select() -> void:
    play_ui_audio(ui_select_audio)

func ui_cancel() -> void:
    play_ui_audio(ui_cancel_audio)

func ui_success() -> void:
    play_ui_audio(ui_success_audio)

func ui_error() -> void:
    play_ui_audio(ui_error_audio)
```

## 5. 为 UI 按钮批量添加音效

### 5.1 批量连接函数

```gdscript
func setup_button_audio(root: Node) -> void:
    for button in root.find_children("*", "Button", true):
        button.focus_entered.connect(ui_focus_changed)
        button.pressed.connect(ui_select)
```

- `find_children("*", "Button", true)` 递归查找所有 `Button` 类型后代节点。

### 5.2 在标题界面和暂停菜单中调用

```gdscript
# TitleScreen.gd
func _ready() -> void:
    Audio.setup_button_audio(self)

# PauseMenu.gd
func _ready() -> void:
    Audio.setup_button_audio(self)
```

### 5.3 注意事项

- `PauseMenu` 的 `Process Mode` 必须设为 `Always`，否则暂停时音效无法播放。
- 该方法目前只处理 `Button`；若需为 `HSlider` 等控件添加音效，可写类似的循环。

## 6. 音乐播放基础

### 6.1 简单播放

```gdscript
func play_music(stream: AudioStream) -> void:
    music_player_1.stream = stream
    music_player_1.play()
```

- 可用于标题界面等不需要过渡的场景。

### 6.2 音乐自动触发器

- 创建脚本 `music_auto_trigger.gd`：

```gdscript
class_name MusicAutoTrigger
extends Node

@export var track: AudioStream
@export var reverb: Audio.ReverbType = Audio.ReverbType.NONE

func _ready() -> void:
    Audio.play_music(track)
    Audio.set_reverb(reverb)
```

- 为脚本添加自定义图标。
- 在需要播放音乐的关卡中添加 `MusicAutoTrigger` 节点，配置 `track`。
- 保存后重新加载项目以显示图标。
