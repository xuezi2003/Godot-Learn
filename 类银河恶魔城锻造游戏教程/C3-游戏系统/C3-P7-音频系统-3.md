# C3-P7：音频系统 - 3

> 本集将暂停菜单中的音量滑块与音频总线连接，实现音量设置的保存与加载，并调整默认音量以避免音乐过响。

## 1. 默认音量调整

### 1.1 问题

- 音乐轨道在游戏中播放时音量过大。
- 若将补间动画终点设为 0.8，会与玩家可配置的 0~1 滑块产生语义冲突。

### 1.2 解决方案

- 保持滑块 0~1 对应 0%~100%。
- 在加载配置时给音乐设置默认 0.8（80%），SFX 与 UI 默认 1.0（100%）。
- 玩家仍可通过滑块自由调整。

## 2. 暂停菜单连接音量滑块

### 2.1 显示系统菜单时初始化滑块

```gdscript
func show_system_menu() -> void:
    pause_screen.visible = false
    system_menu.visible = true
    back_to_map_button.grab_focus()
    
    music_slider.value = AudioServer.get_bus_volume_linear(2)  # Music
    sfx_slider.value = AudioServer.get_bus_volume_linear(3)    # SFX
    ui_slider.value = AudioServer.get_bus_volume_linear(4)     # UI
```

- 总线索引：
  - 0 = Master
  - 1 = EffectPass
  - 2 = Music
  - 3 = SFX
  - 4 = UI

### 2.2 滑块变化回调

```gdscript
func _on_music_slider_changed(value: float) -> void:
    AudioServer.set_bus_volume_linear(2, value)
    SaveManager.save_configuration()

func _on_sfx_slider_changed(value: float) -> void:
    AudioServer.set_bus_volume_linear(3, value)
    Audio.play_spatial_sound(Audio.ui_focus_audio, player_position)
    SaveManager.save_configuration()

func _on_ui_slider_changed(value: float) -> void:
    AudioServer.set_bus_volume_linear(4, value)
    Audio.play_ui_audio(Audio.ui_focus_audio)
    SaveManager.save_configuration()
```

### 2.3 播放测试音

- 调整 SFX 滑块时，在玩家位置播放一个空间音效，便于感受音量大小。
- 调整 UI 滑块时，直接播放一个 UI 音效。
- 需要在暂停菜单脚本中获取玩家位置引用。

```gdscript
var player_position: Vector2

func _ready() -> void:
    var player := get_tree().get_first_node_in_group("player")
    if player:
        player_position = player.global_position
```

### 2.4 连接信号

```gdscript
music_slider.value_changed.connect(_on_music_slider_changed)
sfx_slider.value_changed.connect(_on_sfx_slider_changed)
ui_slider.value_changed.connect(_on_ui_slider_changed)
```

## 3. 保存配置

### 3.1 配置文件路径

- 在 `SaveManager` 中定义：

```gdscript
const CONFIG_PATH := "user://settings.cfg"
var config := ConfigFile.new()
```

### 3.2 保存配置函数

```gdscript
func save_configuration() -> void:
    config.set_value("audio", "music", AudioServer.get_bus_volume_linear(2))
    config.set_value("audio", "sfx", AudioServer.get_bus_volume_linear(3))
    config.set_value("audio", "ui", AudioServer.get_bus_volume_linear(4))
    config.save(CONFIG_PATH)
```

### 3.3 加载配置函数

```gdscript
func load_configuration() -> void:
    var err := config.load(CONFIG_PATH)
    if err != OK:
        # 没有配置文件，设置默认值
        AudioServer.set_bus_volume_linear(2, 0.8)
        AudioServer.set_bus_volume_linear(3, 1.0)
        AudioServer.set_bus_volume_linear(4, 1.0)
        return
    
    AudioServer.set_bus_volume_linear(2, config.get_value("audio", "music", 0.8))
    AudioServer.set_bus_volume_linear(3, config.get_value("audio", "sfx", 1.0))
    AudioServer.set_bus_volume_linear(4, config.get_value("audio", "ui", 1.0))
```

### 3.4 在 SaveManager 就绪时加载

```gdscript
func _ready() -> void:
    load_configuration()
```

### 3.5 配置文件格式

- 保存后会在用户数据目录生成 `settings.cfg`。
- 内容示例：

```ini
[audio]
music=0.4
sfx=1.0
ui=1.0
```

## 4. 缺失配置时的默认值

- 当配置文件不存在时，直接设置默认值。
- 可选：设置默认值后立即调用 `save_configuration()` 生成文件。
- 两种方式皆可，取决于设计需求。

## 5. 其他音效润色

- 暂停游戏时播放一个 UI 音效：

```gdscript
Audio.ui_cancel()  # 或其他合适音效
```

- 从系统菜单返回地图时播放另一个 UI 音效。
- 利用已建立的 `Audio` 系统，一行代码即可完成。

## 6. 测试要点

- 进入暂停菜单 → 系统菜单，滑块位置反映当前音量。
- 拖动滑块时对应总线音量实时变化。
- SFX/UI 滑块调整时播放测试音。
- 关闭游戏后重新打开，音量保持上次设置。
- 删除 `settings.cfg` 后重新启动，音量回到默认值。

## 7. 后续建议

- 将系统菜单做成独立可复用场景，方便在标题画面也使用。
- 添加更多设置项（如全屏、分辨率、语言）。
- 考虑将配置管理拆分为独立的 `ConfigManager` 单例，如果设置项变得非常庞大。
