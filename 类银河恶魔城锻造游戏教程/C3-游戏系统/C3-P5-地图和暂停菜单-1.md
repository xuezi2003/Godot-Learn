# C3-P5：地图和暂停菜单 - 1

> 本集开始构建暂停菜单与系统菜单，并搭建地图系统的 UI 容器，为下一部分的地图节点与玩家位置指示器做准备。

## 1. 暂停菜单场景

### 1.1 创建场景

- 在项目根目录创建 `PauseMenu/` 文件夹。
- 新建场景，根节点类型选择 **CanvasLayer**，命名为 `PauseMenu`。
- 设置：
  - `Layer = 1`（或根据 HUD 层级调整，确保覆盖游戏画面）。
  - `Process Mode = Always`，确保游戏暂停时菜单仍能运行。

### 1.2 附加脚本

- 为 `PauseMenu` 附加脚本 `pause_menu.gd`，保存到 `PauseMenu/Scripts/`。
- 给脚本添加 `class_name PauseMenu`。

### 1.3 控件结构

```
PauseMenu (CanvasLayer)
└── PauseScreenSpace (Control, Full Rect)
    ├── TopPanel (ColorRect)      # 顶部深色条
    │   └── PausedLabel (Label)   # "已暂停"
    ├── PauseScreen (Control)     # 地图与常规暂停项
    │   ├── AbilitiesLabel (Label)
    │   ├── MapLabel (Label)
    │   ├── Map (Control)         # 地图绘制区域
    │   └── SystemMenuButton (Button)
    └── SystemMenu (Control)      # 系统设置界面
        └── VBoxContainer
            ├── HBoxContainer
            │   ├── BackToMapButton (Button)
            │   └── BackToTitleButton (Button)
            ├── Label (Settings)
            ├── HBoxContainer (Music)
            │   ├── Label (Music)
            │   └── MusicSlider (HSlider)
            ├── HBoxContainer (SFX)
            │   ├── Label (SFX)
            │   └── SFXSlider (HSlider)
            └── HBoxContainer (UI)
                ├── Label (UI)
                └── UISlider (HSlider)
```

## 2. 视觉布局

### 2.1 顶部面板

- 使用 `ColorRect`，锚点预设 `Top Wide`。
- 设置最小高度约 36 像素，颜色为黑色或深灰色。
- 添加 `Label`，文本 `已暂停`：
  - 右中对齐，字体放大。
  - 可在字母间加空格或使用 `Font Variation` 的字形间距拉开间距。
  - 颜色设为暗红色/橙色，避免过于抢眼。

### 2.2 暂停主界面

- `PauseScreen` 占满顶部面板下方的空间。
- 左侧留空用于显示能力列表（未来章节解锁）。
- 右侧大区域为 `Map`，用于绘制地图。
- 右下角放置 `SystemMenuButton`，文本 `系统`。

### 2.3 系统菜单

- `SystemMenu` 使用 `VBoxContainer` 居中排列。
- 顶部 `HBoxContainer` 包含：
  - `BackToMapButton`（返回地图）
  - `BackToTitleButton`（返回标题）
- 两个按钮设置 `Container Sizing → Horizontal = Expand`，均分宽度。
- 下方为三个音量滑块，每个滑块前有一个固定宽度标签（60 像素）：
  - 音乐（MusicSlider）
  - 音效（SFXSlider）
  - UI（UISlider）
- 滑块配置：
  - `Min = 0`，`Max = 1`，`Step = 0.05`，`Value = 0.5`。
  - `Container Sizing → Horizontal = Expand`。

### 2.4 间距调整

- 在 `VBoxContainer` 的 `Theme Overrides → Constants → Separation` 中增加间距（例如 20）。
- 可通过添加空 `Control` 节点进一步增大标签与按钮之间的间距。

## 3. 暂停菜单脚本

### 3.1 获取引用

```gdscript
@onready var pause_screen_space: Control = %PauseScreenSpace
@onready var pause_screen: Control = %PauseScreen
@onready var system_menu: Control = %SystemMenu

@onready var system_menu_button: Button = %SystemMenuButton
@onready var back_to_map_button: Button = %BackToMapButton
@onready var back_to_title_button: Button = %BackToTitleButton

@onready var music_slider: HSlider = %MusicSlider
@onready var sfx_slider: HSlider = %SFXSlider
@onready var ui_slider: HSlider = %UISlider
```

### 3.2 显示/隐藏菜单

```gdscript
func show_pause_screen() -> void:
    pause_screen.visible = true
    system_menu.visible = false

func show_system_menu() -> void:
    pause_screen.visible = false
    system_menu.visible = true
    back_to_map_button.grab_focus()
```

### 3.3 按钮连接

```gdscript
func _ready() -> void:
    system_menu_button.pressed.connect(show_system_menu)
    back_to_map_button.pressed.connect(show_pause_screen)
    back_to_title_button.pressed.connect(_on_back_to_title_pressed)

func _on_back_to_title_pressed() -> void:
    get_tree().paused = false
    queue_free()
    SceneManager.transition_scene("标题画面UID", "", Vector2.ZERO, 0)
```

## 4. 玩家脚本中触发暂停

### 4.1 添加暂停动作

- 进入 **项目设置 → 输入映射**，添加 `pause` 动作：
  - 键盘 `Esc`
  - 手柄 `Start` / `Options`

### 4.2 实例化暂停菜单

```gdscript
func _unhandled_input(event: InputEvent) -> void:
    if event.is_action_pressed("pause"):
        get_tree().paused = true
        var pause_menu := preload("res://PauseMenu/PauseMenu.tscn").instantiate()
        add_child(pause_menu)
        get_viewport().set_input_as_handled()
        return
```

- 使用 `get_viewport().set_input_as_handled()` 防止玩家脚本和暂停菜单同时响应同一暂停键，导致“暂停后立即恢复”。

## 5. 焦点控制

### 5.1 进入暂停界面

- 默认不应让焦点跳到 `SystemMenuButton`，因为玩家暂停主要是为了看地图。
- 当玩家按下方向键（右/下）时，再将焦点移动到系统按钮。

### 5.2 方向键聚焦系统按钮

```gdscript
func _unhandled_input(event: InputEvent) -> void:
    if pause_screen.visible:
        if event.is_action_pressed("ui_right") or event.is_action_pressed("ui_down"):
            system_menu_button.grab_focus()
```

### 5.3 进入系统菜单聚焦首项

- 在 `show_system_menu()` 中调用 `back_to_map_button.grab_focus()`，确保进入系统菜单后有焦点。

## 6. UI 输入映射检查

- 打开项目设置中的 `显示内置操作`。
- 确保以下 UI 动作已正确绑定：
  - `ui_accept`：键盘 Enter/Space，手柄确认键。
  - `ui_cancel`：键盘 Esc，手柄取消键。
  - `ui_left` / `ui_right` / `ui_up` / `ui_down`：方向键与手柄方向键/摇杆。
