# C3-P7：音频系统 - 2

> 本集实现音乐的交叉淡入淡出切换、2D 空间音效播放，并将门、开关、保存点等对象的音效接入全局音频系统。

## 1. 音乐交叉淡入淡出

### 1.1 当前轨道索引

```gdscript
var current_track: int = 0
var music_tweens: Array[Tween] = []
```

- `current_track` 为 0 或 1，对应 `MusicPlayer1` 或 `MusicPlayer2`。
- `music_tweens` 用于存储当前正在运行的补间动画，便于在切换前终止。

### 1.2 获取指定音乐播放器

```gdscript
func get_music_player(index: int) -> AudioStreamPlayer:
    if index == 0:
        return music_player_1
    return music_player_2
```

### 1.3 播放音乐（含过渡）

```gdscript
func play_music(stream: AudioStream) -> void:
    var current_player := get_music_player(current_track)
    
    # 如果与当前音乐相同，不重新播放
    if current_player.stream == stream:
        return
    
    # 停止现有补间
    for tween in music_tweens:
        tween.kill()
    music_tweens.clear()
    
    # 计算下一轨道
    var next_track := wrapi(current_track + 1, 0, 2)
    var next_player := get_music_player(next_track)
    
    # 在下一播放器上设置并播放新音乐
    next_player.stream = stream
    next_player.play()
    
    # 淡出当前，淡入下一首
    _fade_out(current_player)
    _fade_in(next_player)
    
    current_track = next_track
```

### 1.4 淡入淡出补间

```gdscript
func _fade_in(player: AudioStreamPlayer) -> void:
    player.volume_linear = 0.0
    var tween := create_tween()
    music_tweens.append(tween)
    tween.tween_property(player, "volume_linear", 1.0, 1.0)

func _fade_out(player: AudioStreamPlayer) -> void:
    var tween := create_tween()
    music_tweens.append(tween)
    tween.tween_property(player, "volume_linear", 0.0, 1.5)
    tween.finished.connect(player.stop)
```

- 淡入时长 1 秒，淡出时长 1.5 秒，形成交叉渐变。
- 使用 `volume_linear` 而不是 `volume_db`，便于在 0~1 线性尺度上操作。

### 1.5 终止旧补间的重要性

- 若玩家在过渡期间频繁切换关卡，旧补间的 `finished.connect(player.stop)` 可能在新音乐已经播放后停止播放器，导致无声。
- 因此切换前必须 `kill()` 所有旧补间并清空数组。

## 2. 2D 空间音效

### 2.1 播放空间音效函数

```gdscript
func play_spatial_sound(stream: AudioStream, pos: Vector2) -> void:
    var player := AudioStreamPlayer2D.new()
    add_child(player)
    player.bus = "SFX"
    player.global_position = pos
    player.stream = stream
    player.finished.connect(player.queue_free)
    player.play()
```

### 2.2 使用场景

- 门撞击、开关触发、敌人受击等世界中发生的事件。
- 调用时传入音效资源与世界位置：

```gdscript
Audio.play_spatial_sound(door_bash_audio, global_position)
```

### 2.3 性能注意事项

- 该方法会动态创建和销毁节点。
- 对大多数游戏来说 Godot 的节点创建/销毁效率足够高。
- 若在每帧生成大量音效（如弹幕游戏），建议使用对象池优化。


### 3.2 开关（Switch）

```gdscript

```

### 3.3 保存点（SavePoint）

```gdscript
# 在玩家交互保存成功时
Audio.ui_success()
```

### 3.4 优势

- 无需在每个对象上添加 `AudioStreamPlayer` 节点。
- 一行代码即可触发音效，总线、位置等细节由音频系统统一处理。
- 通过导出变量可为不同门/开关分配不同音效。

## 4. 混响效果

### 4.1 混响类型枚举

```gdscript
enum ReverbType { NONE, SMALL, MEDIUM, LARGE }
```

### 4.2 设置混响

```gdscript
func set_reverb(type: ReverbType) -> void:
    var reverb := AudioServer.get_bus_effect(1, 0) as AudioEffectReverb
    if reverb == null:
        return
    
    AudioServer.set_bus_effect_enabled(1, 0, true)
    
    match type:
        ReverbType.NONE:
            AudioServer.set_bus_effect_enabled(1, 0, false)
        ReverbType.SMALL:
            reverb.room_size = 0.2
        ReverbType.MEDIUM:
            reverb.room_size = 0.5
        ReverbType.LARGE:
            reverb.room_size = 0.8
```

- 总线索引 `1` 对应 `EffectPass`，效果索引 `0` 对应混响效果器。

### 4.3 在音乐自动触发器中配置

```gdscript
# 在 MusicAutoTrigger._ready() 中
Audio.set_reverb(reverb)
```

- 每个关卡可独立设置混响类型，如小房间用 `SMALL`，大教堂用 `LARGE`。

## 5. 音频总线调试

- 在音频总线面板中可以：
  - **静音（Mute）**某条总线，单独测试音效。
  - **独奏（Solo）**某条总线，只播放该总线声音（注意与 EffectPass 路由结合时可能不直观）。
  - **绕过（Bypass）**效果器。

## 6. 测试要点

- 音乐切换时是否平滑交叉淡入淡出。
- 频繁切换关卡是否不会导致音乐停止。
- 门/开关/保存点是否正确播放对应音效。
- 不同关卡的混响效果是否有明显区别。
- 空间音效是否根据声源位置产生左右声道差异。
