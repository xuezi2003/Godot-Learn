# 09 界面音效与打包发布（完结篇）

## HUD 界面搭建

1. 在 `game` 场景根节点下添加 **Node2D**，命名为 `HUDLayer`。
2. 在 `HUDLayer` 下添加以下节点：

### 生命值显示

- **Sprite2D**，命名为 `LifeIcon`
  - Texture 选择 `props_ui.png`，勾选 Region，裁剪出玩家头像图标。
  - 拖到左上角合适位置。
- **Label**，命名为 `LifeCountLabel`
  - 输入 "×5" 预览效果，实际内容由脚本控制。
  - 在 **Theme Overrides > Font** 中加载像素字体。
  - 调整位置和大小。

### 倒计时显示

- **Sprite2D**，命名为 `TimeIcon`
  - 从 `props_ui.png` 中裁剪出钟表图标。
- **Sprite2D**，命名为 `TimeBar`
  - 从 `props_ui.png` 中裁剪出绿色长条。
  - 通过水平缩放实现进度条效果。

### 游戏结束弹窗

- 在 `game` 根节点下添加 **AcceptDialog**，命名为 `ResultDialog`。
  - Title 设为 `游戏结束`。
  - Size X 设为 200。
  - Flags 中勾选 **Unresizable**。
  - Process > Mode 设为 **When Paused**，确保游戏暂停时仍可交互。
  - 默认隐藏（点击场景树眼睛图标关闭可见性）。

## Game 脚本 UI 逻辑

### 新增变量

```gdscript
const GAME_WIN_TITLE := "胜利！"
const GAME_WIN_MESSAGE := "你成功坚持到了倒计时结束！"
const GAME_LOSE_TITLE := "游戏结束"
const GAME_LOSE_MESSAGE := "你的生命耗尽了，再试一次吧！"

@export var stage_total_time: float = 60.0

@onready var hud_layer: Node2D = $HUDLayer
@onready var life_count_label: Label = $HUDLayer/LifeCountLabel
@onready var time_bar: Sprite2D = $HUDLayer/TimeBar
@onready var result_dialog: AcceptDialog = $ResultDialog

var stage_time_left: float
var time_bar_initial_width: float
var time_bar_initial_position: Vector2
```

### 初始化

```gdscript
func _ready() -> void:
    stage_time_left = stage_total_time
    setup_result_dialog()
    setup_hud()
    scan_spawn_points()
    cache_enemy_configs()
    setup_enemy_spawn_timer()
    spawn_initial_enemies()

func setup_result_dialog() -> void:
    result_dialog.confirmed.connect(_on_result_dialog_closed)
    result_dialog.canceled.connect(_on_result_dialog_closed)
    result_dialog.close_requested.connect(_on_result_dialog_closed)

func setup_hud() -> void:
    time_bar_initial_width = time_bar.texture.get_width()
    time_bar_initial_position = time_bar.position
```

### 更新流程

```gdscript
func _process(delta: float) -> void:
    if result_dialog.visible:
        return

    update_stage_timer(delta)
    update_spawn_interval()
    update_hud()
    check_game_result()

func update_stage_timer(delta: float) -> void:
    stage_time_left -= delta
    stage_time_left = maxf(stage_time_left, 0.0)

func update_hud() -> void:
    life_count_label.text = "×%d" % get_player_current_health()
    update_time_bar()

func update_time_bar() -> void:
    var ratio := stage_time_left / stage_total_time
    time_bar.scale.x = ratio

    # 若时间条居中显示，需调整位置保持左对齐
    var offset := (1.0 - ratio) * time_bar_initial_width / 2.0
    time_bar.position.x = time_bar_initial_position.x - offset

func check_game_result() -> void:
    if stage_time_left <= 0.0:
        show_result_dialog(GAME_WIN_TITLE, GAME_WIN_MESSAGE)
        return
    if get_player_current_health() <= 0:
        show_result_dialog(GAME_LOSE_TITLE, GAME_LOSE_MESSAGE)

func show_result_dialog(title: String, message: String) -> void:
    if result_dialog.visible:
        return

    result_dialog.title = title
    result_dialog.dialog_text = message
    stop_world()
    result_dialog.popup_centered()
    result_dialog.get_ok_button().grab_focus()

func stop_world() -> void:
    enemy_spawn_timer.stop()
    Engine.time_scale = 0.0
    get_tree().paused = true

func _on_result_dialog_closed() -> void:
    get_tree().quit()

func get_player_current_health() -> int:
    if player == null:
        return 0
    return player.get_current_health()
```

### 时间条宽度修正

如果 `TimeBar` 使用整张大图裁剪，通过 `texture.get_width()` 获取的是大图宽度。解决方法：

1. 选中 `TimeBar`。
2. Texture 选择 **AtlasTexture**。
3. 加载 `props_ui.png`，取消 Region 勾选后重新编辑区域选择时间条。
4. 这样 `texture.get_width()` 返回的是实际裁剪后的宽度。

### 刷怪间隔改为倒计时

```gdscript
func _process(delta: float) -> void:
    # ...
    update_spawn_interval()

func get_current_spawn_interval() -> float:
    var ratio := stage_time_left / stage_total_time
    ratio = clampf(ratio, 0.0, 1.0)
    return lerp(end_spawn_interval, start_spawn_interval, ratio)
```

- 倒计时越少，刷新间隔越短，难度越高。

## 音效系统

### 场景音频节点

**game 场景：**

- `AudioContainer`（Node）
  - `BGMPlayer`（AudioStreamPlayer）：背景音乐
  - `WinSFXPlayer`（AudioStreamPlayer）：获胜音效，Process > Mode 设为 **When Paused**
  - `LoseSFXPlayer`（AudioStreamPlayer）：失败音效，Process > Mode 设为 **When Paused**

**player 场景：**

- `AudioContainer`（Node）
  - `ShootSFXPlayer`（AudioStreamPlayer）
  - `MoveSFXPlayer`（AudioStreamPlayer）
  - `PickUpSFXPlayer`（AudioStreamPlayer）

**enemy 场景：**

- `AudioContainer`（Node）
  - `ExplodeSFXPlayer`（AudioStreamPlayer）
  - `HitSFXPlayer`（AudioStreamPlayer）
  - `DeathSFXPlayer`（AudioStreamPlayer）

为各节点选择合适的音频资源，可在 Inspector 中调整音量等参数。

### Game 脚本音频

```gdscript
@onready var bgm_player: AudioStreamPlayer = $AudioContainer/BGMPlayer
@onready var win_sfx_player: AudioStreamPlayer = $AudioContainer/WinSFXPlayer
@onready var lose_sfx_player: AudioStreamPlayer = $AudioContainer/LoseSFXPlayer

func show_result_dialog(title: String, message: String) -> void:
    if result_dialog.visible:
        return
    result_dialog.title = title
    result_dialog.dialog_text = message
    play_result_sfx(title)
    stop_world()
    result_dialog.popup_centered()

func stop_world() -> void:
    enemy_spawn_timer.stop()
    Engine.time_scale = 0.0
    get_tree().paused = true
    player.stop_move_sound()

func play_result_sfx(title: String) -> void:
    bgm_player.stop()
    if title == GAME_WIN_TITLE:
        play_sfx(win_sfx_player)
    else:
        play_sfx(lose_sfx_player)

func play_sfx(player_node: AudioStreamPlayer) -> void:
    if player_node == null or player_node.stream == null:
        return
    player_node.stop()
    player_node.play()
```

### Player 脚本音频

```gdscript
@onready var shoot_sfx_player: AudioStreamPlayer = $AudioContainer/ShootSFXPlayer
@onready var move_sfx_player: AudioStreamPlayer = $AudioContainer/MoveSFXPlayer
@onready var pick_up_sfx_player: AudioStreamPlayer = $AudioContainer/PickUpSFXPlayer

func _physics_process(delta: float) -> void:
    update_invincibility(delta)

    if is_dead:
        velocity = Vector2.ZERO
        set_move_sound_active(false)
        return

    var move_input := Input.get_vector("move_left", "move_right", "move_up", "move_down")
    var is_moving := move_input != Vector2.ZERO
    velocity = move_input * get_effective_move_speed()
    move_and_slide()
    set_move_sound_active(is_moving)

    # ... 射击逻辑

func apply_pickup(config: PickUpConfig) -> bool:
    # ...
    if applied:
        play_sfx(pick_up_sfx_player)
    return applied

func _die() -> void:
    is_dead = true
    velocity = Vector2.ZERO
    set_move_sound_active(false)
    # ...

func stop_move_sound() -> void:
    set_move_sound_active(false)

func set_move_sound_active(active: bool) -> void:
    if active:
        if not move_sfx_player.playing:
            move_sfx_player.play()
    else:
        if move_sfx_player.playing:
            move_sfx_player.stop()

func play_sfx(player_node: AudioStreamPlayer) -> void:
    if player_node == null or player_node.stream == null:
        return
    player_node.stop()
    player_node.play()
```

## 打包发布

1. 打开 **Project > Export**。
2. 点击 **Add**，选择目标平台（如 Windows Desktop）。
3. 若提示模板缺失，点击 **Manage Export Templates > Install from File**，选择第一节下载的导出模板文件。
4. 在导出设置中：
   - **Application > Icon**：选择素材包中的 `icon.png`。
   - 设置导出路径，例如 `release/my_godot_game.exe`。
5. 取消勾选 **Use Debug Export**，点击 **Export Project**。
6. 导出后会生成：
   - `my_godot_game.exe`
   - `my_godot_game.pck`

### 嵌入 PCK

若希望只发布单个文件：

1. 删除之前的 exe 和 pck。
2. 在导出设置中勾选 **Binary Format > Embed PCK**。
3. 重新导出，得到单个 exe 文件。
