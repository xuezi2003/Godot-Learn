# 04 玩家射击系统

## 输入映射

1. 打开 **Project > Project Settings > Input Map**。
2. 添加四个射击方向动作：
   - `shoot_left`：左方向键
   - `shoot_right`：右方向键
   - `shoot_up`：上方向键
   - `shoot_down`：下方向键
3. 斜向射击由代码根据这四个方向组合计算。

## 添加射击冷却计时器

1. 在 `player` 场景中根节点下添加 **Timer** 节点。
2. 命名为 `ShootTimer`。
3. 本视频中不直接使用 `timeout` 信号，而是作为冷却状态计时器查询使用。

## 射击系统需求

- 两种射击模式：
  - **指向性射击**：按方向键向指定方向发射。
  - **螺旋射击**：吃到强化道具后自动向四周旋转发射弹幕。
- 两种模式的子弹数量、冷却时间不同。
- 指向性射击可受射击加速道具影响。
- 角色动画朝向由射击方向优先于移动方向决定，支持倒着走路并向前射击。

## 函数调用链路

```
_physics_process()
├── try_shoot()         # 判断冷却是否结束
│   └── fire_bullets()  # 决定本帧发射几颗子弹
│       └── spawn_bullet(direction)  # 实例化单颗子弹
├── update_facing()     # 根据射击/移动优先级更新朝向
├── update_animation()  # 根据形态和朝向后缀播放动画
└── update_armed_effect()  # 更新浮游炮特效
```

拆分的目的是让输入判断、子弹生成、动画表现相互独立，便于后续扩展更多攻击模式或道具 Buff。

## Player 脚本关键实现

### 常量与变量

```gdscript
const BULLET_SCENE := preload("res://scenes/bullet.tscn")
const ARMED_ANIM_PREFIX := &"armed"
const SPIRAL_ANGLE_STEP := 30.0

@export var fire_interval: float = 0.2
@export var bullet_spawn_distance: float = 12.0

@onready var arm_effect_sprite: AnimatedSprite2D = $ArmEffectSprite
@onready var shoot_timer: Timer = $ShootTimer

var current_fire_rate_multiplier := 1.0
var is_spiral_mode := false
var is_armed_mode := false
var spiral_angle := 0.0
```

- `BULLET_SCENE`：预加载子弹场景，作为实例化模板。
- `bullet_spawn_distance`：子弹出生点相对玩家中心的偏移，模拟从枪口射出。
- `current_fire_rate_multiplier`：当前生效的射速倍率，普通状态为 1.0，武装强化状态可更高。
- `is_spiral_mode` / `is_armed_mode`：强化状态与弹幕模式解耦，后续可独立扩展。

### _ready 初始化

```gdscript
func _ready() -> void:
    shoot_timer.one_shot = true
    shoot_timer.wait_time = get_current_fire_interval()
    update_animation()
    update_armed_effect()
```

### _physics_process

```gdscript
func _physics_process(delta: float) -> void:
    var move_input := Input.get_vector("move_left", "move_right", "move_up", "move_down")
    var shoot_input := Input.get_vector("shoot_left", "shoot_right", "shoot_up", "shoot_down")

    velocity = move_input * move_speed
    move_and_slide()

    if is_spiral_mode:
        try_auto_spiral_shot()
    elif shoot_input != Vector2.ZERO:
        try_shoot(shoot_input)

    update_facing(move_input, shoot_input)
    update_animation()
    update_armed_effect()
```

### 更新朝向

- 射击方向优先于移动方向决定动画朝向。
- 螺旋模式下不读取射击按键，按移动方向更新强化动画朝向。

### try_shoot 射击许可

```gdscript
func try_shoot(shoot_direction: Vector2) -> void:
    if not shoot_timer.is_stopped():
        return

    var direction := shoot_direction.normalized()
    var has_spawned := fire_bullets(direction)
    if has_spawned:
        shoot_timer.wait_time = get_current_fire_interval()
        shoot_timer.start()
```

- 检查 `shoot_timer` 是否停止，只有冷却结束才能发射。
- `fire_bullets` 返回是否成功生成子弹，可用于扩展沉默等效果。

### fire_bullets 分发逻辑

```gdscript
func fire_bullets(direction: Vector2) -> bool:
    if is_spiral_mode:
        spawn_bullet(direction.rotated(deg_to_rad(spiral_angle)))
        spawn_bullet(direction.rotated(deg_to_rad(spiral_angle + 180)))
        spiral_angle += SPIRAL_ANGLE_STEP
        return true
    else:
        return spawn_bullet(direction)
```

### spawn_bullet 实例化子弹

```gdscript
func spawn_bullet(direction: Vector2) -> bool:
    if BULLET_SCENE == null:
        return false

    var bullet := BULLET_SCENE.instantiate() as Bullet
    var root := get_tree().current_scene
    root.add_child(bullet)

    bullet.global_position = global_position + direction * bullet_spawn_distance
    bullet.setup(direction)
    return true
```

- 子弹挂载到当前场景根节点，而不是玩家节点下，避免玩家移动影响子弹位置。
- 使用 `bullet_spawn_distance` 让子弹从角色边缘出生。

### 螺旋射击

```gdscript
func try_auto_spiral_shot() -> void:
    if not shoot_timer.is_stopped():
        return
    fire_bullets(Vector2.RIGHT)
    shoot_timer.wait_time = get_current_fire_interval()
    shoot_timer.start()
```

### 辅助函数

```gdscript
func get_current_fire_interval() -> float:
    return fire_interval / get_fire_rate_multiplier()

func get_fire_rate_multiplier() -> float:
    if is_armed_mode:
        return current_fire_rate_multiplier
    return 1.0

func get_animation_prefix() -> StringName:
    return ARMED_ANIM_PREFIX if is_armed_mode else NORMAL_ANIM_PREFIX
```

### 更新动画

```gdscript
func update_animation() -> void:
    var prefix := get_animation_prefix()
    var animation_name := prefix + "_" + facing_suffix

    if not body_sprite.sprite_frames.has_animation(animation_name):
        animation_name = NORMAL_ANIM_PREFIX + "_" + facing_suffix
        if not body_sprite.sprite_frames.has_animation(animation_name):
            push_warning("动画不存在: " + animation_name)
            return

    if body_sprite.animation != animation_name:
        body_sprite.play(animation_name)
```

- 增加兜底策略：如果找不到当前形态动画，回退到 normal 状态对应朝向动画。

### 更新浮游炮特效

```gdscript
func update_armed_effect() -> void:
    arm_effect_sprite.visible = is_armed_mode
    if is_armed_mode:
        if arm_effect_sprite.sprite_frames.has_animation("default"):
            arm_effect_sprite.play("default")
    else:
        arm_effect_sprite.stop()
```
