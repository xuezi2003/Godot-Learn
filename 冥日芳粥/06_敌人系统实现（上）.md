# 06 敌人系统实现（上）

## 设计思路

敌人系统沿用道具系统的资源驱动思路：

1. **动画资源**：敌人的 SpriteFrames，负责序列帧动画。
2. **配置脚本**：`enemy_config.gd`，定义敌人配置资源的格式。
3. **配置资源**：具体的 `.tres` 文件，填写不同敌人的数值。
4. **场景**：`enemy.tscn`，定义敌人的通用节点结构。
5. **脚本**：`enemy.gd`，处理敌人的通用行为，读取配置资源生效。

## 创建敌人动画资源

1. 在 `resources` 下新建 `animations` 文件夹。
2. 右键新建资源，搜索 **SpriteFrames**。

### enemy_basic（普通敌人）

- 动画：`death`、`move`
- 选择 `enemies.png`，网格 3×6
- `death`：最后一行
- `move`：第二行

### enemy_fast（快速敌人）

- `move`：紫色原始虫所在行

### enemy_tank（带壳敌人）

- `move`：倒数第三行
- 速度慢，生命值高

### enemy_bomber（自爆敌人）

- 动画：`death`、`move`、`explode`
- `death`：倒数第二行
- `move`：第一行
- `explode`：爆炸特效图，网格 5×1，选取所有帧

## 敌人配置脚本

在 `resources/config` 下创建 `enemy_config.gd`，继承 **Resource**：

```gdscript
class_name EnemyConfig
extends Resource

enum EnemyType {
    BASIC,
    TANK,
    FAST,
    BOMBER
}

@export_group("基础信息")
@export var enemy_type: EnemyType
@export var display_name: String

@export_group("基础数值")
@export var max_health: int = 1
@export var move_speed: float = 20.0
@export var collision_radius: float = 6.0

@export_group("动画")
@export var sprite_frames: SpriteFrames
@export var move_animation_name := &"move"
@export var death_animation_name := &"death"

@export_group("死亡效果")
@export var explode_on_death := false
@export var explosion_damage: int = 2
@export var explosion_radius: float = 30.0

@export_group("掉落")
@export var drop_chance := 0.0
@export var possible_drops: Array[PickUpConfig]
```

- 动画名配置为 StringName，运行时动态查找。
- 死亡效果目前用布尔值标记自爆，后续可扩展为枚举。

## 创建敌人场景

1. 在 `scenes` 下新建场景，根节点选择 **CharacterBody2D**。
2. 命名为 `enemy`。
3. 添加子节点：
   - **AnimatedSprite2D**：显示敌人动画。Sprite Frames 可留空，后续由代码根据配置动态赋值。
   - **CollisionShape2D**：敌人身体碰撞外形，Shape 选择 **CircleShape2D**，Radius 4。
   - **Area2D**，命名为 `TouchDamageArea`：敌人伤害检测范围。
     - 子节点 **CollisionShape2D**，Shape 选择 **CircleShape2D**，Radius 6。
   - **Area2D**，命名为 `ExplosionArea`：自爆伤害范围。
     - 子节点 **CollisionShape2D**，Shape 选择 **CircleShape2D**，Radius 8。
4. 为所有 CollisionShape2D 勾选 **Local to Scene**，避免共享形状资源。
5. 可在 Debug Color 中为不同碰撞形状设置不同颜色以便区分。

## 碰撞层级设置

| 节点 | Layer | Mask | 说明 |
|-----|-------|------|------|
| Enemy 根节点 | enemy_body（3） | world（1） | 身体与世界产生物理阻挡 |
| TouchDamageArea | enemy_sensor（4） | player（2）、bullet（5） | 检测与玩家和子弹的碰撞 |
| ExplosionArea | explosion（7） | player（2）、enemy_body（3） | 自爆伤害玩家和周围敌人 |

## 复用闪烁 Shader

1. 选中 `AnimatedSprite2D`。
2. 在 **CanvasItem > Material** 中新建 **ShaderMaterial**。
3. 加载 `blink.gdshader`。
4. 勾选 **Local to Scene**。
